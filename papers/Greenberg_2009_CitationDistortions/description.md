Empirically demonstrates how citation bias, amplification, and invention within a claim-specific citation network convert scientific hypothesis into accepted fact, using the β-amyloid/inclusion body myositis belief system (242 papers, 675 citations) as a case study. Provides a taxonomy of citation distortion types and a computational methodology for detecting them. Directly foundational for the propstore's claim provenance tracking — the distortion taxonomy maps to stance validation rules, and the claim-specific citation network concept is structurally what the propstore implements.
