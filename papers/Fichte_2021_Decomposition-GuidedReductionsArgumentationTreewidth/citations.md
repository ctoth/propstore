## Key Citations for Follow-up

- **[Dung, 1995]** - Foundational paper defining abstract argumentation frameworks and standard semantics. Already in the collection.

- **[Fichte et al., 2019]** - Predecessor work by same group establishing counting complexity and dynamic programming lower bounds for abstract argumentation. Provides the baseline results this paper extends.

- **[Lampis et al., 2018]** - Prior work on QBF reductions preserving treewidth for admissible and preferred semantics. This paper generalizes those results to all standard semantics.

- **[Hecher, 2020]** - Treewidth-aware reductions for answer-set programming that inspired the DG reduction concept. Shows similar techniques apply to ASP.

- **[Besnard and Hunter, 2008]** - Defines the logic-based argumentation framework studied in Section 4 of this paper. Essential reference for understanding ARG, ARG-Check, and ARG-Rel problems.
