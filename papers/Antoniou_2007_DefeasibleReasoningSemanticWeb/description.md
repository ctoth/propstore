Presents DR-Prolog, a Prolog-based system for defeasible reasoning on the Semantic Web. The system translates defeasible theories (facts, strict/defeasible rules, priorities) into logic programs evaluated under Well-Founded Semantics via XSB Prolog. Supports configurable reasoning variants (ambiguity blocking vs. propagating), conflicting literals declarations, NAF simulation, and integration with RDF/RDFS/OWL ontologies. Includes RuleML-compatible syntax and performance benchmarks against Deimos and d-Prolog across structured test theories.
