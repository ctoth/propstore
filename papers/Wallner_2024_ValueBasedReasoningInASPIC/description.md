Extends ASPIC+ with a value-based filtering layer so each agent gets a subjective knowledge base and subjective defeasible rules before argument construction. The key move is that values determine which propositions and rules are even available to an agent, rather than only ranking finished arguments. Useful for propstore whenever different agents should reason from different value-sensitive slices of the same evidence base.
