Introduces a first-order logic of context with `ist(c, p)` and `value(c, e)` as primitives, formalizing enter/exit operations, lifting axioms between contexts, nonmonotonic specialization, and `assuming(c, p)` for natural-deduction-style reasoning.
Applies the formalism to blocks-world lifting, multi-database integration (GE/Navy/AF pricing), planning integration (route + supply planners), question/answer discourse with query/reply axioms, mental-state pedigrees, and the "transcendence" of outer contexts via relative decontextualization.
Primary theoretical source for propstore's context layer — contexts as first-class logical objects qualifying truth, lifting axioms as the discipline for moving facts between branches/sources/render policies without collapsing disagreement.
