Introduces circumscription, a sentence schema for first-order logic that formalizes the conjecture that the entities satisfying a predicate are only those provably required, enabling non-monotonic reasoning without modifying the underlying logic. Defines both predicate circumscription and domain circumscription, proves soundness via minimal model semantics, and demonstrates application to the frame problem in blocks world. Foundational for understanding defeasible reasoning and the retraction of conclusions under new evidence, directly relevant to argumentation frameworks where claims hold unless defeated.
