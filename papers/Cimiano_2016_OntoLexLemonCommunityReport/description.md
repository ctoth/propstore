The W3C OntoLex Community Group's Final Community Report specifying OntoLex-Lemon, an RDF/OWL vocabulary that attaches rich linguistic grounding to ontologies across five modules (ontolex core, synsem, decomp, vartrans, lime).
It defines LexicalEntry/Form/LexicalSense/LexicalConcept with semantics-by-reference, plus syntactic frames, morphological decomposition, lexico-semantic and translation relations, and metadata statistics for lexicalization coverage.
This is the authoritative implementation target for propstore's concept registry: it formalizes the lexicon-ontology interface, condition-annotated senses, and SKOS-distinct lexical concepts that the semantic layer depends on.
