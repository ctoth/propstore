This paper proves that a Bayesian belief network's posterior probability is a linear fractional function (quotient of two linear functions) of any single conditional probability parameter, and defines the sensitivity set — the subset of network nodes whose parameters can affect the query output. These results reduce the computational burden of sensitivity analysis from exhaustive sweeps to 2-3 targeted network evaluations per parameter. Relevant to propstore's probabilistic argumentation layer for understanding how parameter uncertainty propagates through inference.
