Chan and Darwiche define the CD-distance between probability distributions as the log range of probability ratios, proving it is a true metric that provides tight worst-case bounds on how much any odds ratio can change when one distribution replaces another. They apply this to sensitivity analysis in Bayesian networks (where global CD-distance equals local CPT distance) and to belief revision, showing Jeffrey's rule and Pearl's virtual evidence method both minimize CD-distance. The measure is relevant to bounding the impact of parameter perturbations in probabilistic argumentation frameworks.
