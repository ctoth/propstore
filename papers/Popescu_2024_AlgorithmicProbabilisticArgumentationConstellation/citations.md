## Key Citations for Follow-up

- **Li, Oren, and Norman 2011** — Defines the probabilistic argumentation framework (PrAF) model used in this paper. Already collected in propstore as Li_2012 (the 2012 version).
- **Fazzinga, Flesca, and Furfaro 2013, 2015, 2019** — Complexity results for probabilistic abstract argumentation that this paper builds upon. The 2019 paper addresses beyond-independence scenarios relevant to propstore's correlated uncertainty models.
- **Popescu and Wallner 2023** — Predecessor work applying tree-decompositions to assumption-based argumentation (ABA). Direct predecessor to this paper's approach.
- **Samer and Szeider 2010** — Foundational algorithms for propositional model counting via tree-decompositions. The DP approach in this paper builds on these techniques.
- **Hunter and Thimm 2017** — Probabilistic reasoning with abstract argumentation frameworks under the epistemic approach. Already collected in propstore. The contrast between epistemic and constellation approaches is important for understanding which model propstore uses.
