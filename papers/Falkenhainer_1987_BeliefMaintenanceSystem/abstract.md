# Abstract

## Original Text (Verbatim)

This paper addresses the problem of probabilistic reasoning as it applies to Truth Maintenance Systems. A *Belief Maintenance System* has been constructed which manages a current set of probabilistic beliefs in much the same way that a TMS manages a set of true/false beliefs. Such a system may be thought of as a generalization of a Truth Maintenance System. It enables one to reason using normal two or three-valued logic or using probabilistic values to represent partial belief. The design of the Belief Maintenance System is described and some problems are discussed which require further research. Finally, some examples are presented which show the utility of such a system.

---

## Our Interpretation

The paper addresses the gap between binary truth maintenance and probabilistic reasoning by extending the TMS architecture to handle continuous belief intervals via Dempster-Shafer theory. The key contribution is a system that automatically maintains and propagates graded beliefs through a dependency network, combining evidence from multiple sources while reducing to standard TMS when beliefs are absolute. This is relevant to any knowledge management system that must handle uncertain or partial evidence alongside definitive facts.
