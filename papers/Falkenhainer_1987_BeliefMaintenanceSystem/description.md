Presents a Belief Maintenance System (BMS) that generalizes justification-based Truth Maintenance Systems to manage continuous-valued beliefs using Dempster-Shafer theory, automatically propagating belief changes through a dependency network. The system introduces hard and invertible support links, belief-threshold queries, and a rule engine with pattern-directed triggers, reducing to standard TMS behavior when all beliefs are absolute. Directly relevant to propositional stores that must track graded belief strengths, combine evidence from multiple sources, and maintain consistency across dependency networks.
