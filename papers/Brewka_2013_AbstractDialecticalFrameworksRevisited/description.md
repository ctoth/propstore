Repairs and generalizes Abstract Dialectical Framework semantics by defining preferred and stable semantics over three-valued interpretations for arbitrary ADFs rather than only bipolar ones. Adds both static and dynamic preference handling, gives the main complexity results, and sketches the DIAMOND ASP implementation built on Potassco tooling. Directly relevant to propstore as a principled semantics layer for partial-information, preference-sensitive argumentation beyond plain Dung attack graphs.
