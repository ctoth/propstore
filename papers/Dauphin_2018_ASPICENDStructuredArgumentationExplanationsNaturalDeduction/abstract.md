# Abstract

## Paraphrased Abstract
The paper extends ASPIC+ so that structured arguments can contain hypothetical assumptions, proof-by-contradiction arguments, and explanation relations. It then instantiates the framework for semantic-paradox reasoning, using the liar paradox as a simple test case, and proves that the resulting framework satisfies ASPIC+-style rationality conditions together with additional non-triviality conditions needed for paradox applications.

## Interpretation
This is a framework paper with two targets: broaden ASPIC+ to model explanation and natural-deduction-style reasoning, and show that the resulting machinery still has clean metatheoretic properties when deployed on a concrete philosophical-logic example.
