Defines argument strength in abstract argumentation frameworks as the value of a two-person zero-sum game between a proponent and opponent, where the proponent selects admissible sets containing the argument and the opponent selects attacking sets, with payoffs based on relative attack counts.
The game value, computed via linear programming (simplex), yields a strength in [0,1] satisfying nine formal properties including boundedness, monotonicity under attack/counter-attack, and insensitivity to irrelevant information.
Provides a principled gradual semantics grounded in game theory that can rank arguments beyond binary extension membership, directly applicable to the argumentation layer for graded claim resolution.
