# Abstract

## Original Text (Verbatim)

Abstract argumentation (Dung 1995) is a theory of dialectic that allows us to formalise and study various notions of argument acceptability. In this paper, we go one step further and propose a measure of argument strength by applying the concept of value of a game, as defined in Game Theory (von Neumann 1944). The measure thus obtained satisfies a number of intuitively appealing properties that can be derived mathematically from the minimax theorem.

---

## Our Interpretation

The paper addresses the limitation of binary accept/reject semantics in Dung-style argumentation by defining a continuous strength measure via game theory. The key finding is that modeling the proponent-opponent confrontation as a two-person zero-sum game yields a strength value computable by linear programming that satisfies formal monotonicity and sensitivity properties. This is relevant as a principled gradual semantics for ranking competing arguments in abstract argumentation frameworks.
