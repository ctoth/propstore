# Abstract

## Original Text (Verbatim)

We present a set of axioms that justify the use of belief functions to quantify someone's beliefs. We define a credibility function as the function that quantifies the beliefs held by an agent and based on his evidential corpus. Axioms about the credibility function are introduced. They are based on requirements about the impact of changing the algebra on which the credibility function is defined (the so-called coarsening and refinement) and the impact of conditioning. We also introduce a closure axiom which essentially requires that any credibility function defined on an algebra with r+1 atoms can be obtained from credibility functions defined on an algebra with r atoms through refinement, conditioning, and simplex combinations. The axioms entail that the credibility function must be a belief function as introduced by Shafer (1976), giving thus an axiomatic justification for the use of belief functions to quantify beliefs in the transferable belief model. The axiomatization, complete with proofs and many discussed subtleties, is available as a detailed technical report; this paper is a short version.

## Our Interpretation

Smets axiomatizes the credal level of the Transferable Belief Model: starting from a generic point-valued "credibility" function on a propositional space and adding intuitive axioms about doxastic equivalence (A1), uninformative coarsening (Theorem 1), uninformative refinement (R1, R2), conditioning consistency (M1, M2, M3), and a closure axiom, he proves that credibility functions on a finite Boolean algebra are necessarily Dempster-Shafer belief functions, with plausibility functions appearing as duals. The axioms further reject probability theory at the credal level because probability cannot satisfy Gardenfors's homomorphism and preservation simultaneously, and they pin down unnormalized Dempster's rule as the conditioning operation that does. This justifies treating belief functions (and their vacuous element I_Omega) as the right algebra for credal-level uncertainty in systems that need to express honest ignorance distinct from a fabricated probability.
