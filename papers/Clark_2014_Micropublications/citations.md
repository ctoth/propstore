- **Dung 1995 [47]**: Foundation of abstract argumentation frameworks — already implemented in propstore
- **Cayrol & Lagashe-Sches [49]**: Bipolar Abstract Argumentation Systems — directly relevant to SupportGraph/ChallengeGraph structure
- **Verheij 2005 [45, 64]**: Toulmin Argument Model evaluation — the theoretical grounding for the micropublication structure
- **Greenberg 2009 [3]**: Citation distortion analysis — motivating use case for claim network tracing
- **Ciccarese et al. 2012 [71]**: Open Annotation as First-Class Object — the annotation framework underlying the implementation
