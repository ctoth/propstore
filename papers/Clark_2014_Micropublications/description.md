Presents the Micropublications semantic model for representing scientific claims, evidence (data and methods), argumentation structure (support and challenge graphs), attribution, and annotations in biomedical literature. The model is formally defined as MP = (A, mp, c, A_c, Phi, R) and grounded in argumentation theory (Toulmin/Verheij), with OWL ontology implementation using Open Annotation. Directly relevant to propstore's claim representation, bipolar argumentation (SupportGraph/ChallengeGraph maps to Cayrol 2005), concept normalization via Similarity Groups/Holotypes, and provenance-first non-commitment architecture.
