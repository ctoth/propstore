# Citations

## Reference List

[1] R. A. Howard, "Bayesian decision models for system engineering," *IEEE Trans. on Systems Science and Cybernetics*, vol. SSC-1, pp. 36-40, November 1965.

[2] R. A. Howard, "Dynamic inference," *J. Operations Research*, vol. 13, pp. 712-733, September-October 1965.

## Key Citations for Follow-up

- **Howard 1965a (ref [1]):** Bayesian decision models for system engineering — earlier work by the same author laying groundwork for decision-theoretic system analysis. Likely contains the foundational framework that Information Value Theory builds upon.
- **Howard 1965b (ref [2]):** Dynamic inference — the operational inference framework referenced in the expansion concept discussion. May contain the formal treatment of sequential probability updating that underlies the clairvoyance computation.
