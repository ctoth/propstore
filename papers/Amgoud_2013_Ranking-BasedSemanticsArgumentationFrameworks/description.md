Proposes an axiomatic framework for ranking-based argumentation semantics that produce a total preorder over arguments (from most to least acceptable), rather than the binary accept/reject partitions of Dung's extension-based semantics. Introduces eight postulates capturing desirable properties (abstraction, independence, void precedence, defense precedence, counter-transitivity, cardinality precedence, quality precedence, distributed defense precedence) and constructs two concrete semantics --- Discussion-based (Dbs, using linear discussion chains) and Burden-based (Bbs, using iterative burden accumulation) --- proving which postulates each satisfies. Directly relevant to propstore's argumentation layer for computing graded argument acceptability in the render layer's resolution strategies, providing formal correctness criteria (postulates) for any ranking implementation analogous to AGM postulates for belief revision.
