Defines the core model for nanopublications --- minimal citable scientific assertions bundled with provenance and attribution metadata --- and provides a concrete serialization using RDF Named Graphs built entirely on existing Semantic Web standards. Key contributions include a layered ontology (concept, triple, statement, annotation, nanopublication, S-Evidence) and the aggregation mechanism for collecting all evidence about a single statement. Directly relevant to the propstore's claim architecture as a complementary representation for atomic scientific assertions with provenance tracking.
