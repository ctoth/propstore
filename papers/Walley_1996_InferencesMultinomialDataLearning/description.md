Introduces the Imprecise Dirichlet Model (IDM), which uses a set of Dirichlet priors parameterized by a single hyperparameter s to produce initially vacuous upper and lower probabilities from multinomial data that converge as observations accumulate. The IDM uniquely satisfies the embedding, symmetry, and representation invariance principles that no single Bayesian prior can jointly achieve, providing a principled foundation for inference under prior ignorance. Directly grounds the evidence-to-opinion mapping used in subjective logic and propstore's uncertainty representation, where the hyperparameter s corresponds to Josang's non-informativeness prior weight W.
