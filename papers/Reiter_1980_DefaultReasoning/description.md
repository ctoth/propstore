Introduces default logic, a formal framework for reasoning with assumptions that hold "in the absence of information to the contrary," defining default rules with prerequisite/justification/consequent structure, default theories as pairs of defaults and first-order axioms, and extensions as fixed-point belief sets induced by defaults. Key contributions include the identification of normal defaults (a broadly applicable subclass guaranteed to have extensions), a complete proof theory for normal default theories based on top-down linear resolution with indexed clauses, and formal criteria for belief revision when new observations invalidate default assumptions. Directly foundational for the propstore's assumption-tracking and belief-revision architecture --- this is the formal logic that motivates the Truth Maintenance Systems (Doyle 1979, de Kleer 1986) already in the collection.
