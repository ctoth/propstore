# Abstract

## Original Text (Verbatim)

In this paper, the problem of deriving sensible information from a collection of argumentation systems coming from different agents is addressed. The underlying argumentation theory is Dung's one: each argumentation system gives birth to a set of arguments and to the way they interact, which in turn is used to assemble the corresponding acceptable arguments. A key issue is the interaction between arguments which is typically based on a notion of attack; for example, when an argument takes the form of a logical proof, arguments for a statement and arguments against it can be put forward. In that case, the attack relation is related to logical complementarity. This paper addresses merging in Dung's theory of argumentation which includes several systems developed so far for commonsense reasoning or logic programming. It is attached to manage in situations where one assumption on the nature of the arguments or the interactions between arguments has to be done. Instead, the focus is on the general framework. It is proposed to determine the set of acceptable arguments by casting the expansion function which will convey, for each agent, all available information. Three notions of distance are considered. An approach, method which consists in voting on the agents' selected extensions calls for a new approach. To this purpose, a general framework for merging argumentation systems from Dung's theory of argumentation is presented. This objective is achieved through a three step process: first, each argumentation system is expanded into a partial system over the set of all arguments considered by all the agents; then, a set of argumentation systems which are as close as possible to the whole profile is generated; finally, voting is used so that the selected extensions of the resulting systems are in characteristic of the acceptable arguments at the group level.

---

## Our Interpretation

The paper addresses multi-agent argumentation merging: when different agents have different Dung AFs (possibly over different argument sets), how to derive a single merged AF. The key innovation is introducing partial argumentation frameworks (PAFs) with three-valued attack relations to handle incomplete knowledge, then using distance-based merging operators (sum/max/leximax over edit distance) to find AFs closest to the group profile. This is directly relevant to any system that must reconcile conflicting argumentation structures from multiple sources.
