Describes CYC's microtheory system for partitioning knowledge bases into named contexts that scope assertion visibility, enabling contradictory viewpoints to coexist without forcing global consistency.
Demonstrates practical use cases including multiple source viewpoints, temporal progression, and unshared assumptions, with a detailed worked example using contradictions between the Christmas narratives in Matthew and Luke.
Directly relevant to propstore's non-commitment discipline: microtheories are a production implementation of the same contextual partitioning that propstore achieves via ATMS assumption sets, branch isolation, and render-time filtering.
