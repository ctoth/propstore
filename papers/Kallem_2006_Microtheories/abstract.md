# Abstract

## Original Text (Verbatim)

In this paper, we argue that contexts of reasoning are highly relevant to the type of research the Humanities do. We describe the use of contexts in textual analysis. We also discuss some practical considerations in using one implementation of contexts, microtheories in the CYC Project. We conclude that support for contexts is an indispensable part of the ontological toolkit for research in the Humanities.

Keywords: logical contexts, ontologies, CYC

---

## Our Interpretation

This paper addresses the problem of representing contradictory knowledge from multiple sources without forcing global consistency — exactly the "non-commitment discipline" that propstore is built around. CYC's microtheory system partitions assertions into named, hierarchically organized contexts where contradictions are isolated and reasoning remains locally consistent. The key finding is that five distinct use cases (multiple viewpoints, multiple readings, temporal progression, unshared assumptions, and modal reasoning) all map cleanly to the microtheory mechanism, validated through a detailed example of managing contradictions between Matthew's and Luke's Christmas narratives.
