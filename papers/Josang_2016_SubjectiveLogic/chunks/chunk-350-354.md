## Implementation Notes

For propstore, the index of operator entries is the most useful artifact in
this range. It confirms a flat "operators: 92" entry pointing to a master
catalog; if a propstore implementer needs to verify operator coverage against
the book, the canonical list to walk is the one transcribed in the
"Cross-Reference Map (operators -> primary definitional pages)" table above.

Two index entries cross-reference to alternate names that the propstore code
should also recognize as synonyms:
- "UN-AND" is the same operator as "division" (p.110, 128); "UN-OR" is the
  same as "codivision" (p.112). If propstore exposes division/codivision under
  Jøsang names, accepting "UN-AND"/"UN-OR" as aliases matches the book.
- "co-quotient" = "codivision"; "quotient" sits at p.111 as its own entry.
- "pyramid" = "tetrahedron" (p.31), the geometric simplex for trinomial
  opinions.
- "zero belief mass" = "vacuous opinion"; "zero uncertainty" = "dogmatic
  opinion". This nails down the bidirectional mapping the propstore
  honest-ignorance principle relies on: vacuous <=> u=1 <=> all belief mass
  zero; dogmatic <=> u=0.
- "weight: see 'non-informative prior'" — when the propstore code talks about
  the prior weight W, the book's word for it in the index is "non-informative
  prior weight" at pp.27 and 32.
- "prior probability: see 'base rate'" — the book uses "base rate" as the
  canonical term, not "prior probability".
- "state: see 'value'" — value is the canonical term for the elements of a
  domain.
- "ground truth", "subjective truth", and "consensus truth" all redirect to
  the single entry "truth: 209".

The acronym list on pp.333-334 fixes the abbreviations propstore should use if
it adopts Jøsang notation: SL (subjective logic), SN (subjective network), STN
(subjective trust network), SBN (subjective Bayesian network), BRN (base rate
network), IDM (imprecise Dirichlet model), MBR (marginal base rate), PPS
(parallel-path subnetwork), OIS (outbound-inbound set), DAG, DSPG (directed
series-parallel graph), MP, MT, CP, EQU, XOR, ID (inbound degree), OD
(outbound degree).
