## Definitions and Notation

- **Ambiguity vs Vagueness (p.82):** Ambiguity = uncertainty about meaning of statement, multiple plausible interpretations. Vagueness = interpretation clear but values cannot be determined. SL handles vagueness, not ambiguity. Theory of ambiguity outside scope.
- **DST mapping (Eq.5.1, p.84):** Mapping between bba m of DST and belief/uncertainty masses of subjective opinions: `m(x) = b_X(x) ∀x ∈ R(X)`, `m(X) = u_X`. SL cannot assign belief mass to domain X itself; corresponds to (hyper-)Dirichlet model where only observations of values are evidence; X itself cannot be observation.
- **IDM (Imprecise Dirichlet Model, p.85):** non-informative prior weight W=2; upper/lower probabilities by setting max/min base rates.
- **Definition 6.1 (Addition, p.95):** Domain X, x1, x2 disjoint subsets, x1 ∩ x2 = ∅, require x1 ∪ x2 ⊂ X (proper subset). Addition not applicable when X = x1 ∪ x2 (totally vague result; opinion considered vacuous).
- **Definition 6.2 (Subtraction, p.97):** Subset constraint x2 ⊆ (x1 ∪ x2). System must be in state (x1∪x2) whenever in state x2.
- **Definition 6.3 (Complement, p.99):** Domain X = {x, x̄} binary partition; complement operator ¬ is unary. ¬ω_x = ω_x̄.
- **Definition 7.1 (Binomial Multiplication, p.102):** Independent binomial opinions on x ∈ X = {x, x̄} and y ∈ Y = {y, ȳ}; product opinion ω_{x∧y}.
- **Definition 7.2 (Binomial Comultiplication, p.103):** Same setup as 7.1; coproduct ω_{x∨y}.
- **Theorem 7.1 (p.106):** For two binary domains and independent variables with uniform Beta(p,1,1) PDFs, the PDF for the product variable Z=X·Y is `PDF(p(Z=x∧y)) = -ln(p(Z=x∧y))`, for `0 < p(Z) < 1`.
- **Definition 7.3 (Binomial Division, p.110):** Quotient opinion ω_{x∧̃y} from ω_x by ω_y; constraints in Eq.7.13.
- **Definition 7.4 (Binomial Codivision, p.112):** Co-quotient ω_{x∨̃y} from ω_x by ω_y; constraints in Eq.7.16.

## Theorems / Lemmas / Propositions / Corollaries

- **Theorem 7.1 (p.106):** Product of two independent binary uniform variables (Beta(p,1,1)) has PDF `-ln(p)` over `0 < p < 1`. Proof in [51]. Used to compare with Beta(p,1/2,3/2) which is the SL approximation. Expected probabilities equal: E(q_1)=E(Z=x∧y)=1/4. Variance is approximate, projected probability is always exact.

## Operators Defined / Used in This Range

### Addition `+` (p.95–97)
- Symbol `+`. Signature: binomial × binomial → binomial. Domain disjoint subsets x1∩x2=∅; x1∪x2 ⊂ X.
- Eq.(6.1): `b_{x1∪x2} = b_{x1} + b_{x2}`; `d_{x1∪x2} = (a_{x1}(d_{x1}-b_{x2}) + a_{x2}(d_{x2}-b_{x1}))/(a_{x1}+a_{x2})`; `u_{x1∪x2} = (a_{x1} u_{x1} + a_{x2} u_{x2})/(a_{x1}+a_{x2})`; `a_{x1∪x2} = a_{x1}+a_{x2}`.
- Property: P(x1∪x2)=P(x1)+P(x2) (Eq.6.2).
- Note: addition generates vague belief mass from sharp masses; not always intuitive. Different from cumulative fusion (which adds in evidence space, not belief space).

### Subtraction `−` (p.97–98)
- Eq.(6.3): `b_{((x1∪x2)\x2)} = b_{(x1∪x2)} - b_{x2}`; `d` and `u` formulas given (long); `a_{((x1∪x2)\x2)} = a_{(x1∪x2)} - a_{x2}`.
- Constraints (Eq.6.4): `a_{x2} u_{x2} ≤ a_{(x1∪x2)} u_{(x1∪x2)}`; `a_{(x1∪x2)}(d_{(x1∪x2)}+b_{x2}) ≥ a_{x2}(1+b_{x2}-b_{(x1∪x2)}-u_{(x1∪x2)})`.
- Property: P(x1) = P(x1∪x2) − P(x2) (Eq.6.5). Subtraction reduces vagueness; removes vagueness if result singleton.

### Complement `¬` (p.99)
- Unary. Eq.(6.6): `b_{x̄}=d_x; d_{x̄}=b_x; u_{x̄}=u_x; a_{x̄}=1-a_x`.
- Eq.(6.7): `¬ω_x = ω_{x̄}`. Eq.(6.8): `P(¬ω_x) = 1 - P(ω_x)`.

### Binomial Multiplication `·` (p.102)
- Eq.(7.1): 
  - `b_{x∧y} = b_x b_y + ((1-a_x)a_y b_x u_y + a_x(1-a_y)u_x b_y)/(1-a_x a_y)`
  - `d_{x∧y} = d_x + d_y - d_x d_y`
  - `u_{x∧y} = u_x u_y + ((1-a_y)b_x u_y + (1-a_x)u_x b_y)/(1-a_x a_y)`
  - `a_{x∧y} = a_x a_y`
- Property: P(x∧y) = P(x)·P(y) (homomorphism in dogmatic case).

### Binomial Comultiplication `⊔` (p.103)
- Eq.(7.3):
  - `b_{x∨y} = b_x + b_y - b_x b_y`
  - `d_{x∨y} = d_x d_y + (a_x(1-a_y)d_x u_y + (1-a_x)a_y u_x d_y)/(a_x+a_y-a_x a_y)`
  - `u_{x∨y} = u_x u_y + (a_y d_x u_y + a_x u_x d_y)/(a_x+a_y-a_x a_y)`
  - `a_{x∨y} = a_x + a_y - a_x a_y`
- Property: P(x∨y) = P(x) + P(y) − P(x)P(y) (Eq.7.4).
- Variance: Eq.(7.5,7.6) `Var(x∧y) = P·(1-P)·u/(W+u)` and similar for `x∨y`.

### Binomial Division `/` (UN-AND) (p.110-112)
- Eq.(7.12):
  - `b_{x∧̃y} = a_y(b_x + a_x u_x)/((a_y - a_x)(b_y + a_y u_y)) - a_x(1-d_x)/((a_y-a_x)(1-d_y))`
  - `d_{x∧̃y} = (d_x - d_y)/(1 - d_y)`
  - `u_{x∧̃y} = a_y(1-d_x)/((a_y-a_x)(1-d_y)) - a_y(b_x+a_x u_x)/((a_y-a_x)(b_y+a_y u_y))`
  - `a_{x∧̃y} = a_x/a_y`
- Constraints Eq.(7.13): `a_x < a_y`; `d_x ≥ d_y`; `b_x ≥ a_x(1-a_y)(1-d_x)b_y/((1-a_x)a_y(1-d_y))`; `u_x ≥ (1-a_y)(1-d_x)u_y/((1-a_x)(1-d_y))`.
- Property: P(x) = P(x∧y)/P(y).

### Binomial Codivision `⊔̃` (UN-OR) (p.112-113)
- Eq.(7.15):
  - `b_{x∨̃y} = (b_x - b_y)/(1 - b_y)`
  - `d_{x∨̃y} = (1-a_y)(d_x+(1-a_x)u_x)/((a_x-a_y)(d_y+(1-a_y)u_y)) - (1-a_x)(1-b_x)/((a_x-a_y)(1-b_y))`
  - `u_{x∨̃y} = (1-a_y)(1-b_x)/((a_x-a_y)(1-b_y)) - (1-a_y)(d_x+(1-a_x)u_x)/((a_x-a_y)(d_y+(1-a_y)u_y))`
  - `a_{x∨̃y} = (a_x - a_y)/(1 - a_y)`
- Constraints Eq.(7.16): `a_x > a_y`; `b_x ≥ b_y`; `d_x ≥ (1-a_x)a_y(1-b_x)d_y/(a_x(1-a_y)(1-b_y))`; `u_x ≥ a_y(1-b_x)u_y/(a_x(1-b_y))`.
- Property: P(x) = (P(x∨y) - P(y))/(1 - P(y)) (Eq.7.17).

## Implementation Notes

- Subtraction: enforce constraints Eq.6.4 (non-negativity of u and d). Domain check x2 ⊆ (x1 ∪ x2).
- Addition: refuse when x1 ∪ x2 = X (would produce totally vague opinion); treat result as vacuous.
- Division: enforce Eq.7.13 constraints — `a_x < a_y`, `d_x ≥ d_y`, plus inequalities on b_x and u_x. If violated, division is not defined.
- Codivision: enforce Eq.7.16 — `a_x > a_y`, `b_x ≥ b_y`, inequalities on d_x and u_x.
- Multiplication: denominator `(1 - a_x a_y)` — handle case `a_x = a_y = 1` (degenerate, both fully towards x and y → opinion semantics break).
- Variance formulas (7.5,7.6): default W=2; vacuous (u=1, P=0.5) gives variance 1/12 (uniform Beta).
- SL multiplication is APPROXIMATE for Beta PDFs — exact for projected probability, approximate for variance. Worst case: vacuous arguments (Theorem 7.1 illustrates).
- SL multiplication and comultiplication are NOT distributive over each other (Eq.7.7) — only De Morgan duality (Eq.5.12) and self-duality (b↔d, u↔u, a↔1-a, ∧↔∨) hold.
- SL multiplication IS distributive over addition (∪) (Eq.5.11).
- Complement: unary, simple swap. P(¬x) = 1 - P(x).
- Vacuous opinion ω = (0,0,1,1/2) multiplied n times → product opinion still vacuous, but base rate = (1/2)^n → 0; correctly expresses "infinite series of heads → FALSE."
- Reliability analysis for series-parallel networks: serial = AND (multiplication), parallel = OR (comultiplication).
- Provenance: SL operators are well-defined for opinions with both u<1 (uncertain) and u=0 (dogmatic). Dogmatic produces probabilistic-logic homomorphism; vacuous remains vacuous under most operators (with appropriate base-rate updates).
- Probabilistic notation π_x = (P(x), u_x, a_x) is third equivalent representation alongside belief notation (b,d,u,a) and evidence notation (r,s,a). Operators isomorphic across all three.

## Criticisms of Prior Work

- **Kleene's 3-valued logic (p.87):** truth tables produce counter-intuitive results for infinite conjunctions of UNKNOWN values (should be FALSE intuitively but Kleene gives UNKNOWN). SL resolves correctly via base-rate convergence.
- **Dempster-Shafer Theory (p.84):** DST cannot represent base rates; pignistic projection uses default base rates equal to relative cardinalities. SL adds first-class base rate distribution. Also: "different belief fusion situations require different belief fusion operators has often been ignored in the belief theory literature ... [53]". Dempster's rule is one specific fusion operator, not universal.
- **Imprecise Dirichlet Model (Walley) (p.85):** lower/upper probabilities should be interpreted as a rough interval — not literal bounds. Counter-example: 9 red 1 black bag; lower prob of black = 1/3 > actual 0.1.
- **Fuzzy logic (p.86):** handles vague semantic categories; SL handles crisp categories with uncertainty. Different but combinable [61].
- **Bayesian analysis (p.91, p.94):** product of Beta PDFs in pure Bayesian form involves Gauss hypergeometric functions; complex/intractable. SL approximates as Beta PDF — gains tractability at cost of variance approximation.

## Multinomial Multiplication Methods (Ch.8)

Three independent methods for multinomial multiplication that all give same projected probability `P_XY(xy) = P_X(x) P_Y(y)` (Eq.8.8) but differ in resulting uncertainty:

### Common multinomial setup (p.115-117)
- Eq.(8.1): X = {x1,...,x_k} cardinality k; Y = {y1,...,y_l} cardinality l.
- Eq.(8.2): Cartesian product matrix.
- Eq.(8.3): Singleton belief: `b_X(x_i) b_Y(y_j)` for each (x_i, y_j).
- Eq.(8.4): Row belief: `b_X(x_i) u_Y` (rows of X×Y).
- Eq.(8.5): Column belief: `u_X b_Y(y_j)`.
- Eq.(8.6): `u_XY^Domain = u_X u_Y`.
- Eq.(8.7): `a_XY(x_i,y_j) = a_X(x_i) a_Y(y_j)`.
- Eq.(8.8): `P_XY(xy) = P_X(x) P_Y(y)` (projected probability always preserved).
- Eq.(8.9): Multinomial product variance `Var_XY(xy) = P(xy)(1-P(xy)) u_XY / (W + u_XY)`, W=2.

### 8.1.2 Normal Multiplication (p.118-120)

Two-step procedure:
- **Step 1**: Determine normal uncertainty mass.
  - Eq.(8.10): `u_XY^Rows = sum_x b_XY^Rows(x)`; `u_XY^Columns = sum_y b_XY^Columns(y)`.
  - Eq.(8.11): `u_XY^Max = u_XY^Rows + u_XY^Columns + u_XY^Domain`.
  - Eq.(8.12): `u_XY^Min = u_XY^Domain`.
  - Eq.(8.13): `P_X(x) P_Y(y) = (b_X(x) + a_X(x) u_X)(b_Y(y) + a_Y(y) u_Y)`.
  - Eq.(8.14): `P_XY(xy) = b_XY(xy) + a_X(x) a_Y(y) u_XY`.
  - Eq.(8.15): from equating these `u_XY = ((b_X(x)+a_X(x)u_X)(b_Y(y)+a_Y(y)u_Y) - b_XY(xy)) / (a_X(x) a_Y(y))`.
  - Eq.(8.16): For at least one product value, `b_XY^Singletons(x_i y_j) = b_XY(x_i y_j)`.
  - Eq.(8.17): `u_XY^(i,j) = (P_X(x_i) P_Y(y_j) - b_XY^Singletons(x_i y_j)) / (a_X(x_i) a_Y(y_j))` with `u_XY^Min ≤ u_XY^(i,j) ≤ u_XY^Max`.
  - Eq.(8.18): `u_XY = min_{(xy)∈X×Y} [u_XY^(i,j)]` (smallest uncertainty satisfying constraint).
- **Step 2**: Determine normal belief mass.
  - Eq.(8.19): `b_XY(xy) = P_X(x) P_Y(y) - a_X(x) a_Y(y) u_XY = (b_X(x)+a_X(x)u_X)(b_Y(y)+a_Y(y)u_Y) - a_X(x)a_Y(y)u_XY`.
  - Eq.(8.20): Additivity: `u_XY + sum_{(xy)} b_XY(xy) = 1`.
  - Eq.(8.21): `ω_XY = (b_XY, u_XY, a_XY)`.
- Properties: commutative (from Eq.8.19); also associative.
- Generalisation of binomial product (§7.1.1) but binomial has closed expression while multinomial requires stepwise procedure.

### 8.1.3 Justification (p.120)
Requirement: product belief masses must be at least as large as the raw products in Eq.(8.3). Larger uncertainty → smaller belief masses, so requirement constrains how high uncertainty may go. The largest uncertainty satisfying the constraint defines the product uncertainty (= conservative; preserves max possible uncertainty under Eq.8.3 constraint).

### 8.1.4 Proportional Multiplication (p.120-121)
Two-step:
- **Step 1**: Proportional uncertainty.
  - Eq.(8.22): `u_XY / ü_XY = (u_X + u_Y) / (ü_X + ü_Y)` where `ü_X` is theoretical maximum uncertainty mass (Section 3.5.6 — uncertainty-maximised opinion `ω̈_X`).
  - Eq.(8.23): `u_XY = ü_XY (u_X + u_Y) / (ü_X + ü_Y)`.
  - Eq.(8.24): If `(ü_X + ü_Y) = 0` then `u_XY = 0`. Always `(u_X + u_Y) ≤ (ü_X + ü_Y)`, so `u_XY ∈ [0,1]`.
- **Step 2**: Proportional belief.
  - Eq.(8.25): `b_XY(xy) = P_XY(xy) - a_XY(xy) u_XY` for each (xy) in X×Y.
  - Eq.(8.26): `ω_XY = ω_X · ω_Y`.
- Produces slightly less uncertainty than normal (i.e. more aggressive). Differences/similarities not yet fully analysed.

### 8.1.5 Projected Multiplication (p.121)
Hypernomial multiplication then project to multinomial via Eq.(3.30) on p.40.
- Hypernomial product opinion has belief masses `b_XY^Singletons` (Eq.8.3), `b_XY^Rows` (Eq.8.4), `b_XY^Columns` (Eq.8.5), uncertainty `u_XY^Domain` (Eq.8.6), base rate `a_XY` (Eq.8.7).
- Project to multinomial → `ω'_XY` (multinomial product opinion).
- In general projected has less uncertainty than normal product (both same projected probability). Therefore normal/proportional are PREFERRED when significant factor uncertainty exists.

### 8.1.6 Hypernomial Product (p.122-123)
- Hyperdomain R(X) cardinality κ = 2^k − 2; R(Y) cardinality λ = 2^l − 2.
- Cartesian product hyperdomain R(X×Y) cardinality 2^(kl) − 2.
- Hypernomial product has (κλ + κ + λ) belief masses.
- Note `2^(kl) >> 2^(k+l)` for growing k,l → hypernomial products are highly constrained relative to potential hypervalue space.
- Eq.(8.28): Hypervalue belief masses `b_XY^HValues(x_i, y_j) = b_X(x_i) b_Y(y_j)`.
- Eq.(8.29): Hyperrow belief masses `b_XY^HRows = (b_X(x_1)u_Y, ..., b_X(x_κ)u_Y)`.
- Eq.(8.30): Hypercolumn belief masses `b_XY^HColumns = (u_X b_Y(y_1), ..., u_X b_Y(y_λ))`.
- Eq.(8.31): `u_XY^HDomain = u_X u_Y`.
- Eq.(8.32): `b_XY^Hyper = (b_XY^HValues, b_XY^HRows, b_XY^HColumns)`.
- Eq.(8.33): `ω_XY^Hyper = (b_XY^Hyper, u_XY^HDomain, a_XY)`.
- Project to multinomial via Eq.(3.30) → `ω'_XY` (same projected probability as ω_XY^Hyper).

### 8.1.7 Product of Dirichlet PDFs (p.123-124)
- Eq.(8.34): `Dir_XY^e = Dir_X^e · Dir_Y^e`.
- Procedure (Fig.8.2):
  1. Specify (r_X, a_X) and (r_Y, a_Y).
  2. Map Dirichlet PDFs to opinions ω_X, ω_Y via Eq.(3.23).
  3. Compute ω_XY = ω_X · ω_Y (multinomial product, §8.1.2).
  4. Map back to Dir_XY^e via Eq.(3.23).
- HPDF case: project HPDF to hyper-opinion (Eq.3.35), then to multinomial (Eq.3.30), then multiply.
- Approximates Dirichlet convolution; very simple computationally but not equivalent to analytical convolution.

## Additional Implementation Notes

- Three multinomial multiplication methods produce same projected probability but different uncertainty: normal > proportional > projected. Normal preferred when factor uncertainty significant; projected most aggressive.
- Normal multiplication requires iterating over all (xi,yj) ∈ X×Y to find smallest u_XY^(i,j). O(kl) operations.

## §8.3 Multinomial Division (p.128-131+)

### 8.3.1 Elements of Multinomial Division (p.128-129)
- Eq.(8.37): `b_XY` matrix of belief masses on Cartesian product.
- Eq.(8.38): `a_XY` matrix of base rates on Cartesian product.
- Eq.(8.39): `ω_X = ω_XY / ω_Y`.
- **Eq.(8.40) — Unrealistic requirement:** `P_X(x_i) = P_XY(x_i y_1)/P_Y(y_1) = P_XY(x_i y_2)/P_Y(y_2) = ... = P_XY(x_i y_l)/P_Y(y_l)` for all x_i.
  - In general fails — only one specific ω_XY satisfies this for given ω_Y.
- **Eq.(8.41) — Realistic situation:** `P_X(x_i) ≠ P_XY(x_i y_j)/P_Y(y_j)` (in general the ratios differ for different j).
- **Eq.(8.42) — Assumed (for base rates):** `a_X(x_i) = a_XY(x_i y_1)/a_Y(y_1) = ... = a_XY(x_i y_l)/a_Y(y_l)` (does hold for base rates).
- No general analytical solution → two partial methods: averaging proportional division and selective division.

### 8.3.2 Averaging Proportional Division `/` (p.129-131)
- **Eq.(8.43) Limit rule:** if `P_XY(x_i y_j) = P_Y(y_j) = 0` then `P_XY(x_i y_j)/P_Y(y_j) = 0`.
- **Eq.(8.44) Condition:** if `P_XY(x_i y_j) > 0` and `P_Y(y_j) = 0` then no division possible.
- **Eq.(8.45) Preliminary projected probability:** `P_X^Pre(x_i) = (1/l) sum_{j=1}^l P_XY(x_i y_j)/P_Y(y_j)`. Note l = |Y|. Non-additive in general.
- **Eq.(8.46) Normalisation factor:** `v_X^Ave = sum_{i=1}^k P_X^Pre(x_i)`.
- **Eq.(8.47) Average projected probability:** `P_X^Ave(x_i) = P_X^Pre(x_i) / v_X^Ave`.
- **Eq.(8.48):** `P_X^Ave = {P_X^Ave(x_i), for i=1,...,k}`.
- **Eq.(8.49) Inverse property:** `P_X^Ave = (P_X^Ave · P_Y) / P_Y` (round-trip works).
- Disadvantage: in general `P_XY ≠ P_X^Ave · P_Y` (loss).
- Uncertainty heuristic — proportional method:
  - Eq.(8.50): `u_XY/ü_XY = (u_X^Pre + u_Y)/(ü_X + ü_Y) = u_X^Pre/(ü_X + ü_Y) + u_Y/(ü_Y + ü_Y)`. (Note: book equation appears to write u_Y/(ü_Y + ü_Y) — likely typo; structurally proportional average.)
  - Eq.(8.51): `u_X^Pre/(ü_X + ü_Y) = u_XY/ü_XY - u_Y/(ü_X + ü_Y)`.
  - Eq.(8.52): `u_X^Pre = u_XY(ü_X + ü_Y)/ü_XY - u_Y`.
  - Eq.(8.53): If `(u_XY = 0) ∧ (ü_XY = 0)` then `u_XY/ü_XY = 0`.
- **Eq.(8.54) Range constraint:** if `u_X^Pre > 1` then `u_X = 1`; if `u_X^Pre < 0` then `u_X = 0`; else `u_X = u_X^Pre`.
- **Eq.(8.55) Belief masses:** `b_X(x) = P_X(x) - a_X(x) u_X` for each x ∈ X.
- **Eq.(8.56):** `ω_X^Ave = ω_XY / ω_Y`.
- "Averaging" comes from Eq.(8.45); "proportional" comes from Eq.(8.52).

## Implementation Notes — Continued

- **Multinomial division has NO general analytical solution.** Two partial methods: averaging-proportional (round-trip lossy) or selective (requires single observed value).
- Averaging-proportional division: handle 0/0 → 0 (Eq.8.43); positive numerator with zero denominator → division undefined (Eq.8.44).
- Range constraint Eq.(8.54): clip uncertainty to [0,1] after computing u_X^Pre.
- The averaging-proportional quotient does NOT in general satisfy `ω_XY = ω_X · ω_Y` (information lost in division).
- Hypernomial product computation is simpler (no uncertainty-mass synthesis) but produces "vague" belief mass on hypervalues that aren't singletons.
