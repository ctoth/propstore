## Definitions and Notation

- **Theorem 9.1 (Bayes' Theorem with Base Rates)** — `p(x|y) = a(x) p(y|x) / a(y)` where `a(x), a(y)` are base rates (priors). *(p.155, Eq. 9.5)*
- **Theorem 9.2 (Bayes' Theorem with Marginal Base Rate / MBR)** — `p(x|y) = a(x) p(y|x) / [a(x) p(y|x) + a(x̄) p(y|x̄)]`. The MBR `a(y)` is *derived*, not free; this is the only form that survives repeated inversion. *(p.156, Eq. 9.9)*
- **Definition 9.1 (Conditional Deduction with Binomial Opinions)** — Closed-form binomial deduction operator with case analysis (Cases I, II.A.1, II.A.2, II.B.1, II.B.2, III.A.1, III.A.2, III.B.1, III.B.2). *(pp.168-169, Eqs. 9.40-9.51)*
- **Notation `p(y||x)`** — marginal/derived probability of y from law of total probability over x. Double-bar `||` distinguishes derived vs input. *(p.157)*
- **Notation `p(x̃||y)`** — marginal probability of x abduced from evidence on y; tilde `~` indicates inversion via Bayes' theorem. *(pp.157, 158)*
- **Subjective notation (Sec 9.3, p.161):**
  - `ω_X = (b_X, u_X, a_X)`: multinomial opinion on X
  - `b_X(x)`: belief mass for value x
  - `u_X`: uncertainty mass on X
  - `a_X`: base rate distribution over X; `a_X(x)`: base rate of value x
  - `ω_{Y|X}`: SET of multinomial conditional opinions on Y (one per parent value)
  - `ω_{Y|x}`: single multinomial conditional opinion on Y
  - `b_{Y|x}(y)`: conditional belief mass for value y
  - `ω_{X̃|Y}`: set of *inverted* multinomial conditional opinions on X (abduction)
  - `ω_{X̃|y}`: single inverted conditional opinion
  - `b_{X̃|y}(x)`: inverted conditional belief mass
  - `ω_{Y||X}`: deduced multinomial opinion on Y
  - `b_{Y||X}(y)`: deduced belief mass
  - `ω_{X̃||Y}`: abduced multinomial opinion on X
  - `b_{X̃||Y}(x)`: abduced belief mass
  - `⊙`: deduction operator
  - `⊙̃`: abduction operator
  - `φ̃`: conditional inversion operator (subjective Bayes' theorem)
- **Sub-triangle apex opinion `ω_{y||x̂}`**: child opinion resulting from deducing with vacuous antecedent ω̂_x. *(pp.171-172)*
- **Vacuous opinion `ω̂_X = (b_0, 1, a_X)`**: zero-belief mass distribution, uncertainty 1, base rate distribution preserved. *(p.176, Eq. 9.66)*
- **Absolute opinion `ω^i_X = (b^i_X, u^i_X, a_X)`**: `b^i_X(x_i) = 1`, `b^i_X(x_j) = 0 for j≠i`, `u^i_X = 0`. *(p.176)*

## Theorems / Lemmas / Propositions / Corollaries

- **Theorem 9.1 (Bayes' Theorem with Base Rates)** — `p(x|y) = a(x) p(y|x) / a(y)`. Proof via conditional probability `p(y|x) = p(x∧y)/p(x)` reinterpreted with general priors `a(x∧y)/a(x)`. *(p.155)*
- **Theorem 9.2 (Bayes' Theorem with Marginal Base Rate)** — Eq.(9.9). Proof derives `a(y) = a(x)p(y|x) + a(x̄)p(y|x̄)` (law of total probability for base rates). *(p.156)*

## Operators Defined / Used in This Range

### Deduction Operator `⊙` (binomial)
- **Symbol:** `⊙` (circle-dot)
- **Signature:** `ω_{y||x} = ω_x ⊙ (ω_{y|x}, ω_{y|x̄})`
- **Closed-form:** Definition 9.1 (Eq. 9.40-9.51, pp.168-169). Case analysis on relative positions of belief/disbelief in two conditionals + position of `P(y||x̂)` and `P(x)` vs `a_x`.
- **Outputs:**
  - `b_{y||x} = b^I_y - a_y K`
  - `d_{y||x} = d^I_y - (1-a_y) K`
  - `u_{y||x} = u^I_y + K`
  - `a_y` from MBR Eq.(9.36)
- **Intermediate `b^I_y, d^I_y, u^I_y`** — convex combination of conditionals weighted by parent (b_x, d_x, u_x) and parent base rate a_x. *(Eq. 9.41)*
- **Selection coefficient K** — case-dependent; nine cases. *(Eqs. 9.42-9.50)*

### Deduction Operator `⊙` (multinomial)
- **Signature:** `ω_{Y||X} = ω_X ⊙ ω_{Y|X}` *(Eq. 9.30, p.144; reused 9.64 p.176)*
- Maps Ω_X (parent opinion space) to a sub-space of Ω_Y called the **deduction sub-space**. *(p.176)*
- Defined by three constraints:
  - **Constraint 1**: vertices of Ω_X (absolute opinions) map to ω_{Y|x_i}. *(p.176, Eq. 9.65)*
  - **Constraint 2**: apex of Ω_X (vacuous ω̂_X) maps to apex of deduction sub-space ω_{Y||X̂}. *(p.176, Eq. 9.67)*
  - **Constraint 3**: linear projection inside sub-space gives deduced opinion. *(p.176)*

### Abduction Operator `⊙̃` (binomial / multinomial)
- **Symbol:** `⊙̃`
- **Binomial:** `ω_{x̃||y} = ω_y ⊙̃ (ω_{y|x}, ω_{y|x̄}, a_x) = ω_y ⊙ φ̃(ω_{y|x}, ω_{y|x̄}, a_x) = ω_y ⊙ (ω_{x̃|y}, ω_{x̃|ȳ})` *(Eq. 9.28, p.144)*
- **Multinomial:** `ω_{X̃||Y} = ω_Y ⊙̃ (ω_{Y|X}, a_X) = ω_Y ⊙ φ̃(ω_{Y|X}, a_X) = ω_Y ⊙ ω_{X̃|Y}` *(Eq. 9.31, p.145)*
- Composition: invert conditionals via subjective Bayes', then apply deduction.

### Subjective Bayes' Theorem `φ̃`
- **Binomial:** `(ω_{x̃|y}, ω_{x̃|ȳ}) = φ̃(ω_{y|x}, ω_{y|x̄}, a_x)` *(Eq. 9.29, p.144)*
- **Multinomial:** `ω_{X̃|Y} = φ̃(ω_{Y|X}, a_X)` *(Eq. 9.32, p.145)*
- Detailed in Chapter 10.

## Algorithms / Procedures

**Binomial Deduction (Definition 9.1, p.168):**
1. Inputs: ω_x = (b_x, d_x, u_x, a_x); ω_{y|x} = (b_{y|x}, d_{y|x}, u_{y|x}, a_y); ω_{y|x̄} = (b_{y|x̄}, d_{y|x̄}, u_{y|x̄}, a_y).
2. Compute MBR a_y via Eq.(9.36) if u_{y|x}+u_{y|x̄} < 2; else select a_y from [0,1] (free base rate).
3. Compute intermediate b^I_y, d^I_y, u^I_y via Eq.(9.41).
4. Determine selection coefficient K via case analysis (Cases I, II.A.1, II.A.2, II.B.1, II.B.2, III.A.1, III.A.2, III.B.1, III.B.2) in Eqs.(9.42-9.50). Helper P(y||x̂), P(x) by Eq.(9.51).
5. Output: b_{y||x} = b^I_y - a_y K; d_{y||x} = d^I_y - (1-a_y) K; u_{y||x} = u^I_y + K.

**Multinomial Deduction Setup (Sec 9.5.3, p.175-176):**
1. Inputs: ω_X (parent), set ω_{Y|X} = {ω_{Y|x_i} for i=1..k} (conditional opinions), shared base rate a_Y.
2. Compute MBR a_Y via Eq.(9.57) (if not all conditionals vacuous), or use free base-rate intervals Eqs.(9.58, 9.59).
3. Determine projected probabilities P_{Y||X}(y_j) by Eq.(9.61).
4. Construct deduction sub-space using vertices ω_{Y|x_i} (Constraint 1) and apex ω_{Y||X̂} (Constraint 2, vacuous antecedent).
5. Apply linear projection from ω_X inside Ω_X to deduction sub-space (Constraint 3) — full algorithm continues on later pages (177+).

## Implementation Notes

- **MBR validity gate:** Eq.(9.36) requires `u_{y|x} + u_{y|x̄} < 2` (i.e., at least one conditional non-vacuous). If both vacuous (sum = 2), `a_y` is free in [0,1].
- **Dogmatic conditional collapse:** when both `u_{y|x} = u_{y|x̄} = 0`, the free base-rate interval collapses to MBR Eq.(9.39): `a_y = a_x b_{y|x} + a_{x̄} b_{y|x̄}`.
- **Repeated inversion consistency:** must use MBR (Eq. 9.24 multinomial / Eq. 9.36 binomial), not arbitrary base rates, otherwise iterative Bayes' inversion is non-stationary.
- **Nine-case branching for binomial K:** test in order — Case I checks "no flip" between conditionals, then II.A/II.B/III.A/III.B branch on which conditional has higher belief vs. higher disbelief, then sub-case on (P(y||x̂), P(x)) thresholds.
- **Geometric interpretation:** deduced opinion lies inside a sub-triangle in the child opinion space, defined by the three points {ω_{y|x}, ω_{y|x̄}, ω_{y||x̂}}. Sub-triangle collapses to a line when ω_{y||x̂} is collinear with the two conditionals (e.g., uniform sub-conditionals).
- **Sub-triangle apex `ω_{y||x̂}`:** child opinion when antecedent is vacuous; computed independently as a key reference point.
- **Multinomial deduction is iterative/algorithmic, not closed-form:** the binomial closed-form is a special 2x2 case of the more general algorithmic multinomial method (per p.146 and p.152).
- **Vacuous opinion construction:** `ω̂_X = (b_0, 1, a_X)` — preserve base rate, set all belief mass to zero. Implementation: `vacuous(a_X) := MultinomialOpinion(b=zeros, u=1, a=a_X)`.
- **Absolute opinion:** `ω^i_X` has `b^i_X(x_i)=1`, all other belief masses 0, `u^i_X=0`, base rate distribution unchanged.
- **Multinomial MBR Eq.(9.57)** generalizes binomial MBR (9.36); divisor `1 - Σ a_X(x) u_{Y|x}` must be positive (i.e., not all conditionals vacuous).
- **Free base-rate intervals** `[a_Y^-(y), a_Y^+(y)]` per child value y; computed by hypothetically setting a_Y(y) to 0 or 1 in the projected probability formula.

### Section 9.5.4 Method for Multinomial Deduction *(p.177)*

Three-step procedure:
- **Step 1**: Compute MBR distribution `a_Y` per Eq.(9.68):
$$\boldsymbol{a}_Y(y) = \frac{\sum_{x\in\mathbb{X}} \boldsymbol{a}_X(x)\boldsymbol{b}_{Y|x}(y)}{1 - \sum_{x\in\mathbb{X}} \boldsymbol{a}_X(x) u_{Y|x}}$$ *(Eq. 9.68)*
- **Step 2**: Determine sub-simplex apex opinion `ω_{Y||X̂}` *(p.178, Eqs. 9.69-9.74)*. Constraint:
$$\boldsymbol{b}_{Y\|X}(y_j) \geq \min_i [\boldsymbol{b}_{Y|x_i}(y_j)], \quad \forall j=1,\ldots,l$$ *(Eq. 9.69 — Pearl 1990 principle of plausible reasoning [82])*
$$P_{Y\|\hat{X}}(y_j) = \sum_{i=1}^{k} \boldsymbol{a}_X(x_i) P_{Y|x_i}(y_j)$$ *(Eq. 9.70)*
$$P_{Y\|\hat{X}}(y_j) = \boldsymbol{b}_{Y\|\hat{X}}(y_j) + \boldsymbol{a}_Y(y_j) u_{Y\|\hat{X}}$$ *(Eq. 9.71)*
$$u_j = \frac{P_{Y\|\hat{X}}(y_j) - \min_i[\boldsymbol{b}_{Y|x_i}(y_j)]}{\boldsymbol{a}_Y(y_j)}, \quad \forall j=1,\ldots,l$$ *(Eq. 9.72)*
$$u_{Y\|\hat{X}} \leq u_j$$ *(Eq. 9.73)*
$$u_{Y\|\hat{X}} = \min_j[u_j]$$ *(Eq. 9.74 — greatest u satisfying constraints)*
- **Step 3** *(p.179)*: Linear projection of `ω_X` onto deduction sub-space:
$$u_{Y\|X} = u_X u_{Y\|\hat{X}} + \sum_{i=1}^{k} u_{Y|x_i}\boldsymbol{b}_X(x_i)$$ *(Eq. 9.75)*
$$u_{Y\|X} = u_{Y\|\hat{X}} - \sum_{i=1}^{k}(u_{Y\|\hat{X}} - u_{Y|x_i})\boldsymbol{b}_X(x_i)$$ *(Eq. 9.76 — equivalent form)*
$$\boldsymbol{b}_{Y\|X}(y_j) = P_{Y\|X}(y_j) - \boldsymbol{a}_Y(y_j) u_{Y\|X}$$ *(Eq. 9.77)*
$$\omega_{Y\|X} = (\boldsymbol{b}_{Y\|X}, u_{Y\|X}, \boldsymbol{a}_Y)$$ *(Eq. 9.78)*

**Definition 9.2 (Multinomial Conditional Deduction)** *(p.179)*: The opinion `ω_{Y||X}` derived through the 3-step procedure from `ω_{Y|X}` and `ω_X`. Operator symbol `⊙`:
$$\omega_{Y\|X} = \omega_X \odot \boldsymbol{\omega}_{Y|X}$$ *(Eq. 9.79)*

Notes *(p.180)*:
- Special case 2x2 deduction equivalent to Definition 9.1.
- If X is known with `b_X(x_i)=1` (absolute), `ω_{Y||X} = ω_{Y|x_i}`.
- For hyper-opinions: project to multinomial via Section 3.6.2 first, then deduce.

### Chapter 10: Subjective Abduction *(starts p.189 / printed 171)*

#### 10.1 Introduction to Abductive Reasoning *(pp.189-191)*

- Abduction = reasoning opposite to causal conditional direction; from observation to cause.
- Examples: door unlocked (burglar vs. forgot to lock), medical diagnostics, fingerprint matching.
- **AS** = Affected Subjects, **US** = Unaffected Subjects.
- **TP, TN, FP, FN** = True Positives, True Negatives, False Positives, False Negatives.
- AS = TP + FN; US = TN + FP.

$$\text{Sensitivity (TPR)} = \frac{TP}{TP+FN} = \frac{TP}{AS}$$ *(p.190, Eq. 10.1)*

$$\text{Specificity (TNR)} = \frac{TN}{TN+FP} = \frac{TN}{US}$$ *(p.190, Eq. 10.1)*

$$p(\text{'Positive test'} | \text{'Affected subject'}) = TPR \quad\text{(Sensitivity)}$$
$$p(\text{'Positive test'} | \text{'Unaffected subject'}) = 1 - TNR \quad\text{(Unspecificity)}$$ *(p.190, Eq. 10.2)*

- **Base-rate fallacy** (Bar-Hillel et al. [7,66]): erroneously assuming `p(y|x) = p(x|y)`. Approximates well when base rates close to 0.5, fails badly with low base rates and imperfect tests.
- Examples: pregnancy test on a male; **prosecutor's fallacy [88]** in DNA / fingerprint forensics with hundreds of millions of database entries.

#### 10.2 Relevance and Dependence *(p.191)*

- 'Relevance' = conditional relationship between (parent) variable X and a specific value of (child) variable Y.
- 'Dependence' = relationship between variables.
- Continuous measures (not binary).

##### 10.2.1 Relevance and Irrelevance *(pp.192-193)*

**Definition 10.1 (Probabilistic Relevance):**
$$\Psi(y_j|X) = \max_{x_i \in X}[p(y_j|x_i)] - \min_{x_i \in X}[p(y_j|x_i)]$$ *(p.192, Eq. 10.3)*

- Ψ ∈ [0,1]. Ψ=1 → total relevance (determination); Ψ=0 → total irrelevance.

Binary case:
$$\Psi(y|X) = |p(y|x) - p(y|\bar{x})|$$ *(p.192, Eq. 10.4)*

**Definition 10.2 (Subjective Relevance):**
$$\Psi(y_j|X) = \max_{x_i \in X}[P_{Y|x_i}(y_j)] - \min_{x_i \in X}[P_{Y|x_i}(y_j)]$$ *(p.192, Eq. 10.5)*

- Equivalent to **diagnosticity** in **ACH (Analysis of Competing Hypotheses) [34, 5]**.
- **Irrelevance** Ψ̄(y_j|X) = 1 - Ψ(y_j|X) *(Eq. 10.6, p.192)*

##### 10.2.2 Dependence and Independence *(p.193)*

**Definition 10.3 (Dependence):**
$$\delta(Y|X) = \max_{y\in\mathbb{Y}}[\Psi(y|X)]$$ *(p.193, Eq. 10.7)*

- δ(Y|X) ∈ [0,1]. δ=1 total dependence; δ=0 total independence.

**Independence:** δ̄(Y|X) = 1 - δ(Y|X) *(Eq. 10.8, p.193)*

#### 10.3 Binomial Subjective Bayes' Theorem *(pp.193-199+)*

##### 10.3.1 Principles for Inverting Binomial Conditional Opinions *(pp.193-195)*

$$\omega_{x\|y} = \omega_y \odot (\omega_{x|y}, \omega_{x|\bar{y}})$$ *(p.193, Eq. 10.9)*

- Binomial abduction: invert (ω_{y|x}, ω_{y|x̄}) to (ω_{x̃|y}, ω_{x̃|ȳ}), then deduce.
- **Fig. 10.1 (p.194):** Inversion of binomial conditionals. Example with ω_{y|x}=(0.80, 0.20, 0.00, 0.50), ω_{y|x̄}=(0.20, 0.80, 0.00, 0.50), a_x=0.50. Inverted: ω_{x̃|y}=(0.72, 0.12, 0.16, 0.50), ω_{x̃|ȳ}=(0.16, 0.72, 0.12, 0.50). Inversion increases uncertainty.

Projected probabilities:
$$P(y|x) = b_{y|x} + a_y u_{y|x},\quad P(y|\bar{x}) = b_{y|\bar{x}} + a_y u_{y|\bar{x}}$$ *(p.195, Eq. 10.10)*

Inverted projected probabilities (Bayes with MBR):
$$P(x|y) = \frac{a_x P(y|x)}{a_x P(y|x) + a_{\bar{x}} P(y|\bar{x})}$$
$$P(x|\bar{y}) = \frac{a_x P(\bar{y}|x)}{a_x P(\bar{y}|x) + a_{\bar{x}} P(\bar{y}|\bar{x})}$$ *(p.195, Eq. 10.11)*

Dogmatic conditionals (no uncertainty):
$$\underline{\omega}_{x|y} = (P(x|y), P(\bar{x}|y), 0, a_x),\quad \underline{\omega}_{x|\bar{y}} = (P(x|\bar{y}), P(\bar{x}|\bar{y}), 0, a_x)$$ *(p.195, Eq. 10.12)*

##### 10.3.2 Uncertainty Mass of Inverted Binomial Conditionals (4-step procedure) *(pp.195-198)*

Inputs to uncertainty: theoretical maximum uncertainty masses ü_{x|y}, ü_{x|ȳ}; weighted proportional uncertainty u^w_{y|X}; irrelevance Ψ̄(y|X).

**Step 1: Theoretical max uncertainties** ü *(p.196, Eqs. 10.13-10.14):*
$$\text{IF } P(x|y) < a_x \text{ THEN } \ddot{u}_{x|y} = P(x|y)/a_x \text{ ELSE } \ddot{u}_{x|y} = (1-P(x|y))/(1-a_x)$$
Same for ü_{x|ȳ} with P(x|ȳ).

**Step 2: Weighted proportional uncertainty** u^w_{y|X} *(p.196-197, Eqs. 10.15-10.22):*

Sum: $u^S_{y|X} = u_{y|x} + u_{y|\bar{x}}$ *(Eq. 10.15)*

Weights *(Eqs. 10.16, 10.17):*
$$w^u_{y|x} = u_{y|x}/u^S_{y|X} \text{ if } u^S_{y|X}>0 \text{ else } 0$$
$$w^u_{y|\bar{x}} = u_{y|\bar{x}}/u^S_{y|X} \text{ if } u^S_{y|X}>0 \text{ else } 0$$

Theoretical max uncertainties for original conditionals using a_y *(p.197, Eqs. 10.18-10.19):*
- IF P(y|x) < a_y THEN ü_{y|x} = P(y|x)/a_y ELSE ü_{y|x} = (1-P(y|x))/(1-a_y)
- Same for ü_{y|x̄} with P(y|x̄) and a_y.

Weighted proportional uncertainty components *(Eqs. 10.20-10.21):*
$$u^w_{y|x} = w^u_{y|x} u_{y|x}/\ddot{u}_{y|x} \text{ for } \ddot{u}_{y|x}>0; \text{ else } 0$$
$$u^w_{y|\bar{x}} = w^u_{y|\bar{x}} u_{y|\bar{x}}/\ddot{u}_{y|\bar{x}} \text{ for } \ddot{u}_{y|\bar{x}}>0; \text{ else } 0$$

Total weighted proportional uncertainty *(Eq. 10.22):*
$$u^w_{y|X} = u^w_{y|x} + u^w_{y|\bar{x}}$$

**Step 3: Relative uncertainties** ũ_{x|y}, ũ_{x|ȳ} *(p.197, Eq. 10.23):*
$$\widetilde{u}_{x|y} = \widetilde{u}_{x|\bar{y}} = u^w_{y|X} \sqcup \overline{\Psi}(y|X) = u^w_{y|X} + \overline{\Psi}(y|X) - u^w_{y|X}\overline{\Psi}(y|X)$$
where ⊔ denotes the **coproduct operator**.
- Note: in binomial case, ũ_{x|y} = ũ_{x|ȳ}.
- Increases as either weighted proportional uncertainty u^w_{y|X} or irrelevance Ψ̄(y|X) increases. "Alice ignorant about whether Bob carries an umbrella in sunny or rainy weather" example.

**Step 4: Inverted binomial uncertainties** *(p.198, Eq. 10.24):*
$$u_{x\widetilde{|}y} = \ddot{u}_{x|y}\, \widetilde{u}_{x|y}$$
$$u_{x\widetilde{|}\bar{y}} = \ddot{u}_{x|\bar{y}}\, \widetilde{u}_{x|\bar{y}}$$

##### 10.3.3 Deriving Binomial Inverted Conditionals *(p.198)*

$$\omega_{x\widetilde{|}y} = \begin{cases} b_{x\widetilde{|}y} = P(x|y) - a_x u_{x\widetilde{|}y} \\ d_{x\widetilde{|}y} = 1 - b_{x\widetilde{|}y} - u_{x\widetilde{|}y} \\ u_{x\widetilde{|}y} \\ a_x \end{cases}$$ *(Eq. 10.25)*

Same form for ω_{x̃|ȳ} *(Eq. 10.26)*.

**Definition 10.4 (Binomial Subjective Bayes' Theorem)** *(p.199):*
$$(\omega_{x\widetilde{|}y}, \omega_{x\widetilde{|}\bar{y}}) = \widetilde{\phi}(\omega_{y|x}, \omega_{y|\bar{x}}, a_x)$$ *(Eq. 10.27)*

Equivalent to multinomial Bayes' (Definition 10.6) in the binary case.

##### 10.3.4 Convergence of Repeated Inversions *(p.199)*

Double inversion with MBR returns original projected probabilities:
$$(p(y|x), p(y|\bar{x})) = \widetilde{\phi}(p(x|y), p(x|\bar{y}), a(y)) = \widetilde{\phi}(\widetilde{\phi}(p(y|x), p(y|\bar{x}), a(x)), a(y))$$ *(p.199, Eq. 10.28)*

- Projected probabilities preserved under repeated inversion (when MBRs used).
- Uncertainty mass *grows* under repeated inversions, converging toward theoretical maxima.
- Fig. 10.3 / Table 10.1 (forward refs to p.200+) illustrate convergence with example a_x=a_y=0.50.

## CONSOLIDATED IMPLEMENTATION NOTES (additional)

- **Coproduct operator ⊔:** `a ⊔ b = a + b - a·b` (used in Eq. 10.23). Equivalent to probabilistic OR / inclusion-exclusion for two events.
- **Inversion four-step algorithm (Sec 10.3.2):** sequence is deterministic — compute ü, then weights, then ü on original, then weighted proportional, then coproduct with irrelevance, then multiply.
- **Uncertainty-maximised conditional ω̈:** preserves projected probability P(x|y) but maximizes u; computed by setting either b or d to zero depending on whether P(x|y) is below or above base rate a_x.
- **Vacuous opinion = uniform Beta:** ω = (0, 0, 1, a) corresponds to Beta(p, W·a, W·(1-a)) with default W=2, a=0.5 → Beta(p,1,1) = uniform. The term 'Any' in the augmented truth table corresponds to this opinion.
- **Three-valued logic mapping:** TRUE → (1,0,0,a), FALSE → (0,1,0,a), UNCERTAIN → (0,0,1,1/2) — Kleene-style logic.
- **Repeated inversion uncertainty growth:** projected probabilities are stationary under repeated φ̃ when MBR is used; uncertainty masses are monotone non-decreasing and converge to theoretical maxima ü.
- **Double inversion identity:** φ̃ ∘ φ̃ on projected probabilities is the identity (with MBRs) but is *not* the identity on full opinions (uncertainty grows).
- **Bayes' theorem with MBR (Eq. 9.9 / 9.24)** is required for repeated-inversion consistency. With arbitrary base rate, conditionals drift after multiple inversions.
- **Diagnosticity ↔ Relevance Ψ:** conceptual identity with ACH-style intelligence analysis (Heuer [34], etc.).
