# Abstract

## Original Text (Verbatim)

This paper explains the importance of classifying argumentation schemes, and outlines how the classification system can be used as a tool for building argument mining tools that identify argument schemes and fit them to given natural language text in a corpus. It uses argumentation schemes that are set out as defeasible argument forms with a named set of critical questions matching each scheme, and that have a well-defined premise-conclusion structure used to identify arguments fitting a given scheme. It provides a survey of the literature on scheme classification. What are to be regarded much as templates at a set of the most widely used argumentation schemes are surveyed and explained systematically, including some that are difficult to classify. A new classification system showing the currently most important schemes is presented.

## Our Interpretation

The paper addresses the fragmentation of argumentation scheme taxonomies by proposing a unified hierarchical classification system. Its key contribution is a tree-structured taxonomy (Figure 4) that organizes approximately 28 argumentation schemes by source-dependency, reasoning type, and inferential structure, with formal premise-conclusion representations for each. This is relevant for computational argumentation systems that need to automatically identify and classify argument types in natural language text.
