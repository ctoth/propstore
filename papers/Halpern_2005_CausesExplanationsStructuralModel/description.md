Provides a formal definition of actual causality using structural equation models, handling classic problem cases (preemption, overdetermination) via counterfactual reasoning with contingent interventions.
The definition (AC1-AC3) requires factual truth, a counterfactual witness under some partition of variables, and minimality — resolving difficulties that plagued Lewis-style counterfactual accounts.
Relevant to propstore's argumentation layer as a rigorous framework for modeling causal arguments, and its model-relativity aligns with the non-commitment principle of holding multiple rival causal models without forcing canonical status.
