Defines a formal framework for logic-based argumentation where arguments are minimal consistent subsets of a knowledge base paired with their deductive consequences, with formal characterizations of defeaters, undercuts, rebuttals, canonical undercuts, argument trees, and argument structures. Introduces the categoriser/accumulator framework for aggregating competing arguments, and shows that binary argumentation, counting-based approaches, argumentation logic, and Dung's abstract framework are special cases or related systems. Directly relevant to propstore's argumentation layer as it provides the precise logical definitions for constructing arguments from claims and determining when they conflict.
