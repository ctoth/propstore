Presents Lenticular Lenses, a framework for constructing context-specific equality links between Semantic Web datasets instead of globally collapsing resources through `owl:sameAs`.
Defines linksets, lenses, property-specific indiscernibility/propagation contexts, singleton-property metadata, and set-like lens operators over candidate correspondences.
Directly relevant to propstore's concept reconciliation discipline because it treats identity as a provenance-rich, task-specific view rather than a canonical storage-time merge.
