Presents Defeasible Logic Programming (DeLP), a formalism combining logic programming with defeasible argumentation where tentative information is represented as defeasible rules, arguments are built by chaining rules, and conflicts are resolved through dialectical tree analysis with a modular comparison criterion. The system provides four-valued query answers (YES/NO/UNDECIDED/UNKNOWN) and prevents pathological argumentation patterns through formal acceptability conditions on argumentation lines. Directly relevant to propstore's argumentation layer as the closest existing system to quantified rule-based reasoning with dialectical analysis.
