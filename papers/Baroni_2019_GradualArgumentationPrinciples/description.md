A systematic unification of fine-grained properties for gradual argumentation semantics into 11 Groups of Principles (GP1-GP11), parameterized across three framework types (aQBAF, sQBAF, QBAF). Shows that two overarching meta-principles — balance and monotonicity — subsume all 11 GPs. Classifies 15+ existing gradual semantics (h-categorizer, Euler-based, DF-QuAD, CMA, complete labeling, etc.) against this principled taxonomy, revealing which semantics satisfy which principles and under what parameter choices.
