Altmejd et al. train machine learning models on 131 replication attempts from four large-scale projects to predict binary replication success and relative effect sizes, achieving 70% accuracy (AUC 0.77) using basic statistical features of original studies. The key predictors are original effect size, p-value, sample size, and whether the effect is a main effect or interaction, with out-of-sample validation on SSRP confirming 71% accuracy (AUC 0.73). This provides empirical base rates and feature importance rankings directly applicable to calibrating belief strength and uncertainty in propstore's argumentation layer when evaluating competing empirical claims.
