## Key Citations for Follow-up

- Halpin and Hayes (2008): Provides the four owl:sameAs usage categories that McCusker and McGuinness map into the Identity Ontology.
- Ding et al. (2010): Empirical evidence about owl:sameAs usage in Linked Data, useful for understanding real-world misuse.
- OWL Web Ontology Language Reference (2004): Defines owl:sameAs and the strict identity semantics this paper critiques.
- SKOS Simple Knowledge Organization System Reference (2009): Supplies skos:exactMatch, skos:closeMatch, and skos:related, which are mapped into the proposed identity hierarchy.
- OWL 2 Direct Semantics (2009): Supplies the property-chain mechanism used to reconstruct selective isomorphism.
