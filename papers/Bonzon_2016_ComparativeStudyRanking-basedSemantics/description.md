Provides a systematic axiomatic comparison of five ranking-based semantics for abstract argumentation (Categoriser, Discussion-based, Burden-based, Matt & Toni, Tuples*), defining 16 desirable properties and determining which semantics satisfy which properties via formal proofs. The central result is a property-satisfaction matrix showing that no single semantics satisfies all properties, revealing fundamental behavioral differences between approaches. Directly relevant for choosing which gradual semantics to implement in the propstore's argumentation layer.
