# Abstract

## Original Text (Verbatim)

Argumentation is a process of evaluating and comparing a set of arguments. A way to compare them consists in using a ranking-based semantics which rank-orders arguments from the most to the least acceptable ones. Recently, a number of such semantics have been proposed independently, often associated with some desirable properties. However, there is no comparative study which takes a broader perspective. This is what we propose in this work. We study the existing ranking-based semantics in the literature discussing on the semantics that returns a unique ranking between arguments in the light of the proposed properties. That allows us to underline the differences of behavior between the existing semantics, and to propose a better reading of the different choices one has on this matter.

---

## Our Interpretation

This paper addresses the lack of systematic comparison between independently proposed ranking-based semantics for abstract argumentation. The key contribution is a comprehensive property-satisfaction table showing which of 16 desirable axioms each of five semantics (Categoriser, Discussion-based, Burden-based, Matt & Toni, Tuples*) satisfies. This is directly relevant for choosing which gradual semantics to implement in an argumentation system, as no single semantics satisfies all properties.
