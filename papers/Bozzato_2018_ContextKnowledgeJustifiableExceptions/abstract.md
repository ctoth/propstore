# Abstract

## Original Text (Verbatim)

Dealing with context dependent knowledge is a well known area of study that roots in John McCarthy's seminal work. More recently, the Contextualized Knowledge Repository (CKR) framework has been proposed as a logic based approach in the context of the Semantic Web, providing a two layered structure, modeled by a global context and a set of local contexts. The global context not only contains the meta knowledge defining the properties of local contexts, but also holds the global (context independent) object knowledge that is shared by all of the local contexts. In many practical cases, however, it is desirable to leave the possibility to "override" the global object knowledge at the local level; in other words, it is interesting to recognize the pieces of knowledge that can admit exceptional instances in the local contexts that do not need to satisfy the general axioms. To this end, we introduce a CKR framework where the global knowledge is extended with defeasible axioms that can be included in the global context. The latter are verified in the local contexts only for the instances for which no exception is verifiable, where exceptions require a justification in terms of facts that are provable from the knowledge base. We formally define this semantics and study some semantic and computational properties, where we characterize the complexity of the major reasoning tasks, among them satisfiability testing, instance checking, and conjunctive query answering. Furthermore, we present a translation of enriched CKRs with knowledge bases in the Description Logic SROIQ-RL under the novel semantics to datalog programs under the stable model (answer set) semantics. We also present an implementation prototype and examine its performance in experiments with synthetic benchmarks of different sizes and degrees of defeasibility for overridings. Finally we compare our representation approach with some major formalisms for expressing defeasible knowledge in Description Logic and contextual knowledge representation (the work adds to the body of results on using deductive database technology such as SQL and datalog in these areas, and provides an expressiveness formalism in terms of (data complexity) for exception handling by overriding).

---

## Our Interpretation

This paper extends the CKR framework with defeasible axioms that allow local contexts to override global knowledge when justified by local evidence (clashing assumption sets). The key contribution is a complete, sound, and correct datalog translation enabling reasoning via answer set programming, with tractable model checking and coNP-complete entailment. This is directly relevant to propstore's need for contextual defeasible reasoning where global claims can be overridden by local evidence without collapsing disagreement at storage time.
