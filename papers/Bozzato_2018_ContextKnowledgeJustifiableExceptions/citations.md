## Key Citations for Follow-up

- **[31] Brewka, Eiter (2007)** - Equilibria in heterogeneous nonmonotonic multi-context systems. The foundational work on MCS that CKR extends; essential for understanding the multi-context framework and how CKR's coverage relation differs from MCS bridge rules.

- **[25] Bikakis, Antoniou (2010)** - Defeasible contextual reasoning with arguments. An alternative approach using argumentation semantics for defeasible MCS, with local preference orderings on contexts. Direct comparison point for CKR's clashing assumption approach.

- **[18] Giordano et al. (2009)** - ALC+T: a preferential extension of description logics. The main alternative for defeasible reasoning in DLs using typicality operators, but without contextual structure. Important contrast with CKR's modular approach.

- **[42] Bonatti et al. (2015)** - A new semantics for overriding in description logics. Class-level defeasibility via normality concepts — CKR explicitly argues for instance-level granularity over this approach.

- **[12] Serafini, Homola (2012)** - Contextualized knowledge repositories for the semantic web. The predecessor CKR paper (monotonic version) that this paper extends with defeasible axioms. Essential background.
