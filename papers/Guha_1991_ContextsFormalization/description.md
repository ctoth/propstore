Guha's 1991 Stanford PhD thesis formalizes McCarthy's informal `ist(c, p)` context proposal as first-class rich objects in first-order logic, giving contexts syntax, semantics, proof theory, and a Default Coreference Rule (DCR) with abnormality predicates; it then walks through 21 applications (microtheories, problem solving contexts, hypothetical reasoning, fixTime/fixTimeType, perspectives, polysemy, granularity, database integration) grounded in Cyc. The thesis contributes the primary operationalization of context that propstore cites and directly underwrites propstore's formal non-commitment discipline: lifting IS relative decontextualization, defaults are blockable, and the render layer — not storage — decides which contextual rival wins. This paper is foundational for propstore's concept/semantic layer; every context-qualified claim, every microtheory-style source branch, and every articulation axiom between heterogeneous paper-local vocabularies traces back to constructs Guha introduces here.
