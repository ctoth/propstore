# Abstract

## Original Text (Verbatim)

This book is not a traditional textbook. No single course could cover all of this material. But if you build AI reasoning systems, or want to learn how, this book provides the background, techniques, and hands-on experience you need. The programs described in this book implement many of the most important AI problem-solving techniques, including pattern-directed inference, truth maintenance, constraint languages, and qualitative reasoning. They provide a set of building blocks for constructing more complex systems, as well as a shared infrastructure for experimentation and research.

---

## Our Interpretation

Building Problem Solvers teaches the implementation of core AI reasoning infrastructure — truth maintenance systems (JTMS, LTMS, ATMS), pattern-directed inference engines, constraint propagation languages, and qualitative reasoning — through working Common Lisp code. It provides complete, buildable implementations of each system along with the theory behind them, organized as composable building blocks. The book is directly relevant to argumentation and belief revision systems since it covers the foundational dependency-tracking and assumption-management machinery (TMS family) that underlies modern nonmonotonic reasoning.
