Presents a tutorial introduction to the ASPIC+ framework for structured argumentation, which constructs arguments from strict and defeasible inference rules applied to knowledge bases, defines three attack types (undermining, rebutting, undercutting), and generates Dung-style abstract argumentation frameworks with preference-based defeat.
The framework satisfies key rationality postulates (sub-argument closure, consistency) under specified conditions and can model diverse approaches including Walton-style argument schemes and assumption-based argumentation.
Directly relevant as a formal foundation for modeling conflicts between claims in a knowledge store, where claims map to premises, inference rules capture claim relationships, and Dung semantics determine which claims are jointly justified.
