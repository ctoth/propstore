## Key Citations for Follow-up

1. **Dung (1995)** - The foundational work on abstract argumentation that ASPIC+ builds upon. Essential for understanding the semantics (grounded, preferred, stable extensions) that ASPIC+ generates.

2. **Modgil & Prakken (2013)** - The full technical version of ASPIC+ with complete proofs, general contrariness function, and detailed rationality postulate analysis. The authoritative reference for the framework.

3. **Caminada & Amgoud (2007)** - Defines the rationality postulates (sub-argument closure, consistency) that ASPIC+ is designed to satisfy. Key for understanding what properties a correct implementation must have.

4. **Toni (2014)** - Tutorial on Assumption-Based Argumentation (ABA), a closely related framework. Useful for comparing approaches and understanding when ABA vs ASPIC+ is more appropriate.

5. **Prakken (2010)** - Earlier version of ASPIC+ (then called ASPIC). Shows the evolution of the framework and contains some additional examples and discussion.
