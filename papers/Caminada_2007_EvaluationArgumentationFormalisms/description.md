Defines three rationality postulates (closure, direct consistency, indirect consistency) for evaluating rule-based argumentation systems, demonstrates through concrete counterexamples that existing formalisms including ASPIC violate them, and provides two closure-operator solutions (propositional closure and transposition closure with restricted rebutting) that guarantee all three postulates under Dung's standard semantics.
Key contributions include the postulate framework itself, which became the standard evaluation criteria for structured argumentation systems, and the transposition closure + restricted rebutting solution adopted by ASPIC+ (Modgil & Prakken 2014, 2018).
Directly foundational for the propstore's argumentation architecture, providing the formal correctness criteria that any ASPIC+-based claim evaluation system must satisfy.
