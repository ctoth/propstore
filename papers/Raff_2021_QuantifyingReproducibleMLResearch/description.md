This paper presents the first quantitative survival analysis of ML paper reproducibility, studying 255 papers with Cox proportional hazard models and XGBoost+SHAP to identify which paper-level features predict independent reproduction success. The key finding is that paper readability (Cix metric) and pseudo code availability are the strongest predictors of reproducibility, while code availability alone is not statistically significant. The empirical methodology and feature importance results are relevant to propstore's reasoning about evidence quality and claim reliability assessment.
