Presents an end-to-end pipeline for parsing argumentation structures in persuasive essays, combining token-level sequence labeling for argument component identification with SVM classification for component types and relations, globally optimized by an integer linear programming (ILP) joint model that enforces tree structure constraints. Introduces a corpus of 402 persuasive essays annotated with discourse-level argumentation structures (major claims, claims, premises, support/attack relations) achieving substantial inter-annotator agreement, and demonstrates that the joint model significantly outperforms pipeline approaches and heuristic baselines across all subtasks. Directly relevant for the propstore's claim extraction and argument mining capabilities, providing empirical grounding for argumentation structure patterns and indicator lexicons that can inform how claims and their relations are identified in text.
