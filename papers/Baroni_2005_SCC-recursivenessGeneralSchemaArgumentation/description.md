Baroni and Giacomin propose a general recursive schema for argumentation semantics based on decomposing an argumentation framework into its strongly connected components (SCCs). The schema is parameterized by a "base function" that determines local extension computation within each SCC, enabling a unified treatment of existing semantics (grounded, preferred) and novel ones. They introduce CF2 semantics, which uses maximal conflict-free sets as the base function and correctly handles odd-length attack cycles and self-defeating arguments — cases where preferred semantics produces only the empty extension.
