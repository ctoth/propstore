## Key Citations for Follow-up

1. **Baroni & Giacomin 2003** — The earlier ECSQARU paper that first introduced SCC decomposition for argumentation. Precursor to this work; contains initial formal results on odd-length cycles.

2. **Dung 1995** — The foundational paper for abstract argumentation frameworks. Essential background; already in propstore's reference set.

3. **Pollock 1992** — Defeasible reasoning; relevant to propstore's defeat handling and the distinction between rebutting and undercutting defeat.

4. **Prakken & Sartor 1997** — Argument-based logic programming with priorities; connects the abstract AF level to structured argumentation (ASPIC+ lineage). Relevant for propstore's ASPIC+ integration.

5. **Baroni & Giacomin 2004 / 2007 (AIJ)** — The full journal version of this workshop paper. Contains complete proofs, additional semantics (AD1, AD2, CF1, CF2), and comprehensive comparison. Should be the primary reference for implementation.
