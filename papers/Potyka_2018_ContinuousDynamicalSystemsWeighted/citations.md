## Key Citations for Follow-up

- **Amgoud and Ben-Naim 2017** — Defines the Euler-based semantics and axiomatic framework that this paper directly extends and improves upon. Essential for understanding the baseline models.
- **Rago et al. 2016 (DF-QuAD)** — The Discontinuity-free QuAD model that is one of the main comparison targets. Already in propstore collection.
- **Cayrol and Lagasquie-Schiex 2005** — Foundational work on gradual valuation in bipolar argumentation frameworks. Defines the bipolar AF structure used throughout.
- **Baroni et al. 2019 (From fine-grained properties to broad principles)** — Broader axiomatic framework for gradual argumentation that contextualizes this paper's property analysis.
- **Mossakowski and Neuhaus 2018** — Modular semantics approach at the same venue (KR 2018), potentially complementary to this paper's continuous dynamics approach.
