Ghidini and Giunchiglia (2001) define Local Models Semantics (LMS), a formal semantic framework for contextual reasoning built on two principles: locality (reasoning uses only part of what is available, constituting a "context") and compatibility (reasoning across contexts must be mutually consistent). The framework uses families of first-order languages, each with its own local models, connected via compatibility sequences and compatibility relations that constrain which combinations of local models can coexist. A model is a non-empty compatibility relation excluding the all-empty sequence. Contexts are defined as sets of local models within compatibility sequences---partial objects rather than complete possible worlds. Logical consequence is defined for labelled formulae (i:phi), where assumptions in one context prune compatibility sequences affecting what follows in other contexts.

The proof theory uses Multi-Context (MC) systems: sets of axiomatic formal systems connected by bridge rules that propagate consequences across languages. Internal rules operate within a single language/context; bridge rules connect different contexts. The paper validates LMS through two extended examples: (1) reasoning with viewpoints (the "magic box" scenario where two observers see partial views of a box, with compatibility constraints linking their observations), formalized as chain models with soundness/completeness for the MC system MV; and (2) reasoning about belief (hierarchical chains of belief contexts for an agent reasoning about its own beliefs), formalized as HMB models with bridge rules Rdw (downward reflection) and Rupr (upward reflection with restrictions), yielding systems equivalent to modal logic K when combined (MBK). Full soundness and completeness proofs are provided for both examples via canonical model constructions.

The paper compares LMS with possible worlds semantics, noting that LMS avoids the assumption of a single global language and allows local satisfiability independent of the model structure. It also compares with Guha's context formalization and other approaches. The framework has been applied to distributed first-order logics for heterogeneous knowledge base federation, multi-agent systems, and agent belief modeling.
