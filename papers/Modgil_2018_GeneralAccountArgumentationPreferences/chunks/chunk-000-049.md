## Key Definitions

### Definition 1 (Dung AF)
A Dung argumentation framework (AF) is a tuple $\langle \mathcal{A}, \mathcal{C} \rangle$ where $\mathcal{C} \subseteq \mathcal{A} \times \mathcal{A}$. A is a set of arguments, C is an attack/conflict relation. Status evaluated as extensions.

### Definition 1 (cont.) — Extensions
For $T \in \{complete, preferred, grounded, stable\}$: $T$ is acceptable/conflict-free/admissible. Standard Dung semantics definitions.

### Definition 2 (ASPIC+ argumentation system)
An argumentation system is a tuple $\langle \mathcal{L}, \overline{\cdot}, \mathcal{R}, n \rangle$ where:
- $\mathcal{L}$ is a logical language
- $\overline{\cdot}$ is a function from $\mathcal{L}$ to $2^\mathcal{L}$ (contrariness function)
- $n$ is a naming convention for defeasible rules
- $\mathcal{R} = \mathcal{R}_s \cup \mathcal{R}_d$ — strict ($\mathcal{R}_s$) and defeasible ($\mathcal{R}_d$) inference rules
- Each $c \in \mathcal{L}$ has at most one contrarity
- Rules of form $\phi_1, \ldots, \phi_n \to \phi$ (strict) or $\phi_1, \ldots, \phi_n \Rightarrow \phi$ (defeasible)
- $\overline{\varphi}$ denotes contraries/contradictories

### Definition 3 (Consistency)
For any $S \subseteq \mathcal{L}$: in the closure of $S$ under strict rules, denoted $Cl_{\mathcal{R}_s}(S)$:
- **directly consistent** iff $\nexists \varphi, \psi \in S$ such that $\varphi \in \overline{\psi}$
- **indirectly consistent** iff $Cl_{\mathcal{R}_s}(S)$ is directly consistent

### Definition 4 (ASPIC+ knowledge base)
A knowledge base in an argumentation system $\langle \mathcal{L}, \overline{\cdot}, \mathcal{R}, n \rangle$ is $\mathcal{K} = \mathcal{K}_n \cup \mathcal{K}_p$ consisting of two disjoint subsets: $\mathcal{K}_n$ (axiom/certain premises) and $\mathcal{K}_p$ (ordinary premises).

### Definition 5 (ASPIC+ arguments)
An argument $A$ on the basis of a knowledge base $\mathcal{K}$ in an argumentation system $\langle \mathcal{L}, \overline{\cdot}, \mathcal{R}, n \rangle$ is:
1. $\varphi$ if $\varphi \in \mathcal{K}$ with: $Prem(A) = \{\varphi\}$; $Conc(A) = \varphi$; $Sub(A) = \{\varphi\}$; $DefRules(A) = \emptyset$; $Rules(A) = \emptyset$; $TopRule(A) = undefined$
2. $A_1, \ldots, A_n \to \psi$ if $A_1, \ldots, A_n$ are arguments such that there exists a strict rule $Conc(A_1), \ldots, Conc(A_n) \to \psi$ in $\mathcal{R}_s$
3. $A_1, \ldots, A_n \Rightarrow \psi$ if $A_1, \ldots, A_n$ are arguments such that there exists a defeasible rule $Conc(A_1), \ldots, Conc(A_n) \Rightarrow \psi$ in $\mathcal{R}_d$

Functions: $Prem(A)$, $Conc(A)$, $Sub(A)$, $DefRules(A)$, $Rules(A)$, $TopRule(A)$, $DirectSub(A)$.

### Notation 2 (Notation for Arguments)
- $S \vdash \psi$: there exists a strict argument $A$ such that $Conc(A) = \psi$ with all premises taken from $S$
- Arguments written as lists of premises and rules separated by semicolons
- $||F||$ notation for a set of arguments

### Definition 6 (c-consistent)
A set $S \subseteq \mathcal{L}$ is c-consistent if for no $\varphi$ it holds that $S \vdash \varphi, \overline{\varphi}$. Otherwise $S$ is c-inconsistent. $S$ is minimally c-inconsistent iff $S$ is c-inconsistent and $\forall S' \subset S$, $S'$ is c-consistent.

### Definition 7 (c-consistent argument)
An argument $A$ on the basis of a knowledge base $\mathcal{K}$ in an argumentation system $\langle \mathcal{L}, \overline{\cdot}, \mathcal{R}, n \rangle$ is c-consistent iff $Prem(A)$ is c-consistent.

### Definition 8 (ASPIC+ attacks)
$A$ attacks $B$ iff $A$ undercuts, rebuts, or undermines $B$:
- **Undermining**: $A$ undermines $B$ (on $B'$) iff $Conc(A) \in \overline{\psi}$ for some $B' \in Sub(B)$ such that $B'$ is an ordinary premise $\psi$
- **Rebutting**: $A$ rebuts $B$ (on $B'$) iff $Conc(A) \in \overline{\psi}$ for some $B' \in Sub(B)$ of the form $B'_1, \ldots, B'_n \Rightarrow \psi$ (i.e., $B'$ has a defeasible top rule with conclusion $\psi$, and $Conc(A)$ is a contrary of $\psi$)
- **Undercutting**: $A$ undercuts $B$ (on $B'$) iff $Conc(A) \in \overline{n(r)}$ for some $B' \in Sub(B)$ such that $B'$ has defeasible top rule $r$; equivalently $Conc(A) = \neg n(r)$ where $n(r)$ names the rule

### Notation 5
$\rightarrow$ denotes the attack relation, $\Rightarrow$ the defeat relation.

### Definition 9 (ASPIC+ defeat)
Let $A$ attack $B$ on $B'$. $A$ is said to:
- **successfully rebut** (i.e., rebut-defeat) $B$ iff $A$ rebuts $B$ on $B'$, and $A \not\prec B'$
- **successfully undermine** (i.e., undermine-defeat) $B$ iff $A$ undermines $B$ on $B'$, and $A \not\prec B'$
- **successfully undercut** (i.e., undercut-defeat = undercut-attack): always succeeds (undercutting is preference-independent)
$A$ defeats $B$ iff $A$ rebut-defeats, undermine-defeats, or undercut-defeats $B$.

### Definition 10 (Argumentation theory)
An argumentation theory is a tuple $AT = \langle \mathcal{AS}, \mathcal{K} \rangle$ where $\mathcal{AS}$ is an argumentation system and $\mathcal{K}$ is a knowledge base in $\mathcal{AS}$.

### Definition 11 (c-Structured Argumentation Frameworks)
Let $AT$ be an argumentation theory $\langle \mathcal{AS}, \mathcal{K} \rangle$:
- A *structured argumentation framework (SAF)* defined by $AT$ is a triple $\langle \mathcal{A}, \mathcal{C}, \preceq \rangle$ where $\mathcal{A}$ is the set of all finite arguments constructed from $\mathcal{K}$ in $\mathcal{AS}$, $\mathcal{C}$ is the attack relation, and $\langle \mathcal{A}, \mathcal{V} \rangle \subseteq \mathcal{C}$ iff $X$ attacks $Y$.
- A *c-structured argumentation framework (c-SAF)* defined by $AT$ is a triple $\langle \mathcal{A}, \mathcal{C}, \preceq \rangle$ where $\mathcal{A}$ is the set of all c-consistent finite arguments from $\mathcal{K}$ in $\mathcal{AS}$, $\preceq$ is an ordering on $\mathcal{A}$.

### Definition 12 (Well-defined (c-)SAF)
Let $AT = \langle \mathcal{AS}, \mathcal{R}, n \rangle$ be an argumentation theory, $AT = \langle \mathcal{AS}, \mathcal{K} \rangle$. We say that $AT$ is:
- **closed under contraposition** iff for any minimal $S \subseteq \mathcal{L}$, $L \in S$ and $S \vdash \psi$, then $S \setminus \{L\} \cup \{\overline{\psi}\} \vdash \overline{L}$ (with appropriate modifications)
- **closed under transposition** iff if $\phi_1, \ldots, \phi_n \to \psi \in \mathcal{R}_s$, then for $i = 1 \ldots n$, $\phi_1, \ldots, \phi_{i-1}, \overline{\psi}, \phi_{i+1}, \ldots, \phi_n \to \overline{\phi_i} \in \mathcal{R}_s$

### Definition 13 (Defeat conflict free for (c-)SAFs)
Let $S \subseteq \mathcal{A}$: $S$ is defeat-conflict-free iff $\nexists A, B \in S$ such that $A$ defeats $B$.

### Definition 14 (Attack conflict free for (c-)SAFs)
Let $S \subseteq \mathcal{A}$: $S$ is attack-conflict-free iff $\nexists A, B \in S$ such that $A$ attacks $B$. (Strictly stronger than defeat-conflict-free.)

### Definition 15 (Extensions and justified arguments)
In either case, justified arguments are evaluated on the basis of extensions of Dung framework $\langle \mathcal{A}, \mathcal{D} \rangle$ as defined in Definition 1. For $T \in \{admissible, complete, preferred, grounded, stable\}$:
- $\varphi$ is $T$-sceptically justified iff for all $T$ extensions $E$, there exists $A \in E$ such that $Conc(A) = \varphi$
- $S$ is a $T$ extension if $S$ is defined in defeat-conflict-free (or attack-conflict-free) terms, and an att-$T$ extension if $S$ is defined as attack-conflict-free.

### Definition 19 (Orderings $\lhd_s$)
Let $\Gamma$ and $\Gamma'$ be finite sets. Then $\lhd_s$ is defined as:
1. If $\Gamma = \emptyset$ then $\Gamma \not\lhd_s \Gamma'$
2. If $\Gamma' = \emptyset$ and $\Gamma \neq \emptyset$ then $\Gamma \lhd_s \Gamma'$
3. If $s = \text{Eli}$: $\Gamma \lhd_{\text{Eli}} \Gamma'$ if $\exists X \in \Gamma$ s.t. $\forall Y \in \Gamma', X < Y$
4. If $s = \text{Dem}$: $\Gamma \lhd_{\text{Dem}} \Gamma'$ if $\forall X \in \Gamma, \exists Y \in \Gamma', X < Y$

Then $\Gamma \leq_s \Gamma'$ iff $\Gamma \lhd_s \Gamma'$ or $\Gamma' = \Gamma$.

### Definition 20 (Last-link principle)
Let $s \in \{\text{Eli}, \text{Dem}\}$. Then $B \prec A$ under the last-link principle iff:
1. $\text{LastDefRules}(B) \lhd_s \text{LastDefRules}(A)$; or
2. $\text{LastDefRules}(B) = \emptyset$, $\text{LastDefRules}(A) = \emptyset$, and $\text{Prem}_p(B) \lhd_s \text{Prem}_p(A)$

Then $B \preceq A$ iff $B \prec A$ or, if $\text{LastDefRules}(A) \neq \emptyset$ then $\text{LastDefRules}(A) = \text{LastDefRules}(B)$, else $\text{Prem}_p(A) = \text{Prem}_p(B)$.

### Definition 21 (Weakest-link principle)
Let $s \in \{\text{Eli}, \text{Dem}\}$. Then $B \prec A$ under the weakest-link principle iff:
1. If both $B$ and $A$ are strict, then $\text{Prem}_p(B) \lhd_s \text{Prem}_p(A)$; else;
2. If both $B$ and $A$ are firm, then $\text{DefRules}(B) \lhd_s \text{DefRules}(A)$; else;
3. $\text{Prem}_p(B) \lhd_s \text{Prem}_p(A)$ and $\text{DefRules}(B) \lhd_s \text{DefRules}(A)$

Then $B \preceq A$ iff $B \prec A$ or, $\text{DefRules}(A) = \text{DefRules}(B)$ and $\text{Prem}_p(A) = \text{Prem}_p(B)$.

### Definition 22 (Reasonable inducing orderings)
$\preceq$ is said to be *reasonable inducing* if it is a strict partial ordering (irreflexive and transitive) and for any $s \in \{\text{LastDefRules}, \text{DefRules}, \text{Prem}_p\}$, for all arguments $D_1, \ldots, D_k, A$ such that $\{D_1, \ldots, D_k\} \neq \{A\}$, it is not the case that for some $s' = \{i \mid 1 \leq i \leq n\}$, $D_i \lhd_s A$.

### Definition 23 (Abstract Logic)
An abstract logic is a pair $\langle \mathcal{L}, cn \rangle$, where $\mathcal{L}$ is a language and the *consequence operator* $cn$ is a function from $2^\mathcal{L}$ to $2^\mathcal{L}$ satisfying:
1. $S \subseteq cn(S)$
2. $cn(cn(S)) = cn(S)$
3. $cn(S) = \bigcup_{S' \subseteq_{\text{fin}} S} cn(S')$
4. $cn(\{p\}) \neq \mathcal{L}$ for some $p \in \mathcal{L}$
5. $cn(\emptyset) \neq \mathcal{L}$

Here $Y \subseteq_f X$ means that $Y$ is a finite subset of $X$. A set $S \subseteq \mathcal{L}$ is defined as consistent if $cn(S) \neq \mathcal{L}$, and as inconsistent otherwise.

### Definition 24 (Arguments and attacks in abstract logic)
Let $\langle \mathcal{L}, cn \rangle$ be an abstract logic and $\langle \mathcal{L}, \preceq \rangle$ a theory in $\langle \mathcal{L}, cn \rangle$, where $\preceq \subseteq \mathcal{L} \times \mathcal{L}$ and $\preceq$ a preordering over $\Sigma$:
- an AL-argument is a pair $\langle \Gamma, \varphi \rangle$ such that: (1) $S \subseteq \Sigma$, (2) $S$ is consistent, (3) $\varphi \in cn(\Gamma)$, (4) no proper subset of $S$ satisfies (1-3)
- $\langle \Gamma, \varphi \rangle$ AL-undermines $\langle \Gamma', \psi \rangle$ if there exists $q \in \Gamma'$ such that $\{p, q\}$ is inconsistent

### Definition 25 (AL c-SAF based on abstract logic with preferences)
$\Delta = \langle \mathcal{A}, \mathcal{C}, \preceq \rangle$ is defined as an AL c-SAF based on an AL argumentation theory based on $\langle \mathcal{L}, cn \rangle$ and $\langle \Sigma, \preceq \rangle$, where:
- $\mathcal{L} = \mathcal{L}'$ (same language)
- $\mathcal{R}_s = \emptyset$, and for all $S \subseteq \mathcal{L}$ and $\varphi \in \mathcal{L}$, $S \to \varphi \in \mathcal{L}$ iff $\varphi \in cn(S)$
- $\mathcal{K}_n = \emptyset$
- Contrariness: $\overline{\varphi}$ is nonempty for all $\varphi$
- $\preceq$ is an argument ordering satisfying certain conditions

### Definition 27 (Premise minimal ASPIC+ arguments)
Let for any argument $A$, $A_-$ be any argument such that $Prem(A_-) \subseteq Prem(A)$ and $Conc(A_-) = Conc(A)$. Given a set of ASPIC+ arguments $\mathcal{A}$, let $\mathcal{A}_- = \{A \in \mathcal{A} \mid$ there is no $A_- \in \mathcal{A}$ such that $Prem(A_-) \subset Prem(A)\}$ be the premise minimal arguments in $\mathcal{A}$.

### Definition 28 (Preferred subtheory / Brewka)
A *default theory* $T$ is a tuple $\langle T_1, \ldots, T_n \rangle$, where each $T_i$ is a set of formulae in a classical first-order language $\mathcal{L}'$. A preferred subtheory is a set $S = S_1 \cup \ldots \cup S_n$ with $S_i \subseteq T_i$ and $S$ is a maximal consistent subset with ordering respected.

### Definition 18 (Reasonable Argument Orderings)
An argument ordering $\preceq$ is *reasonable* iff:
1. i) $\forall A, B$, if $A$ is strict and firm and $B$ is plausible or defeasible, then $B \prec A$;
   ii) $\forall A, B$, if $B$ is strict and firm then $B \not\prec A$;
   iii) $\forall A, A', B$ such that $A'$ is a strict continuation of $\{A\}$, if $A \not\prec B$ then $A' \not\prec B$, and if $B \not\prec A$ then $B \not\prec A'$ (strict continuation preserves preference)
2. Asymmetry strengthening: for any $\{C_1, \ldots, C_n\}$ finite subset of $\mathcal{A}$, and for $i = 1 \ldots n$, let $C^{+\setminus i}$ be some strict continuation of $\{C_1, \ldots, C_{i-1}, C_{i+1}, \ldots, C_n\}$. Then it cannot be that for all $i$, $C' \prec C_i$ where $C'$ is a strict continuation of $\mathcal{A} \setminus C_i$.

## Key Theorems and Proofs

### Proposition 9
Let $\langle \mathcal{A}, \mathcal{C} \rangle$ be a c-SAF. If $A_1, \ldots, A_n$ are acceptable w.r.t. some conflict-free $E \subseteq \mathcal{A}$, then $\{A_1, \ldots, Prem(A_n)\}$ is c-consistent.

### Proposition 10
Proposition 9 implies that Dung's fundamental lemma holds.

### Proposition 11
Let $\mathcal{A}, \mathcal{A}'$ be acceptable w.r.t. an admissible extension $S$ of a (c-)SAF $\langle \mathcal{A}, \mathcal{C} \rangle$. Then:
1. $S' = S \cup \{A\}$ is admissible
2. $A'$ is acceptable w.r.t. $S'$

(This ensures that given reasonable argument orderings, a well-defined c-SAF satisfies Dung's fundamental lemma, and admissible extensions form a complete partial order.)

### Theorem 12 (Sub-argument Closure)
Let $\Delta = \langle \mathcal{A}, \mathcal{C}, \preceq \rangle$ be a (c-)SAF and $E$ an att-complete extension of $\Delta$. Then for all $A \in E$: if $A' \in Sub(A)$ then $A' \in E$.

### Theorem 13 (Closure under Strict Rules)
Let $\Delta = \langle \mathcal{A}, \mathcal{C}, \preceq \rangle$ be a c-SAF and $E$ an att-complete extension of $\Delta$. Then $\{Conc(A) | A \in E\} = Cl_{\mathcal{R}_s}(\{Conc(A) | A \in E\})$.

### Theorem 14 (Direct Consistency)
Let $\Delta = \langle \mathcal{A}, \mathcal{C}, \preceq \rangle$ be a (c-)SAF and $E$ an att-admissible extension of $\Delta$. Then $\{Conc(A) | A \in E\}$ is consistent.

### Theorem 15 (Indirect Consistency)
Let $\Delta = \langle \mathcal{A}, \mathcal{C}, \preceq \rangle$ be a (c-)SAF and $E$ an att-complete extension of $\Delta$. Then $Cl_{\mathcal{R}_s}(\{Conc(A) | A \in E\})$ is consistent.

### Proposition 19
Let $\preceq$ be defined according to the last link principle, based on a reasonable inducing $\leq_p$. Then $\preceq$ is reasonable.

### Proposition 20
Let $\preceq$ be defined according to the weakest link principle, based on a reasonable inducing $\leq_p$. Then $\preceq$ is reasonable.

### Proposition 21

### Proposition 23

### Proposition 26
Let $\langle \mathcal{AS}, \mathcal{K} \rangle$ be based on $\langle \mathcal{L}', cn \rangle$ and $\langle \Sigma, \preceq \rangle$. Let $\Delta_1$ be the c-SAF defined by $\langle \mathcal{AS}, \mathcal{K} \rangle$ and $\Delta_2$ the AL c-SAF defined by $\langle \mathcal{AS}, \mathcal{K} \rangle$. Then, for $T \in \{complete, preferred, grounded, stable\}$: $E$ is a $T$ extension of $\Delta_1$ iff $E$ is a $T$ extension of $\Delta_2$.

### Proposition 27
Let $\langle \mathcal{AS}, \mathcal{K} \rangle$ be based on $\langle \mathcal{L}', cn \rangle$ and $\langle \Sigma, \preceq \rangle$. Then $A$ is a c-consistent premise minimal argument on the basis of $\langle \mathcal{AS}, \mathcal{K} \rangle$ iff $\langle Prem(A), Conc(A) \rangle$ is an abstract logic argument on the basis of $\langle \Sigma, \preceq \rangle$.

### Proposition 28
Let $\Delta$ be the (c-)SAF $\langle \mathcal{A}, \mathcal{C}, \preceq \rangle$ defined on the basis of an AT for which $\preceq$ is defined such that for any $A, A' \in \mathcal{A}$: if $A \neq A', A \preceq B$. ### Proposition 30

### Proposition 32

### Corollary 17 (Strict Continuation of Arguments)

### Lemma 41 (Appendix, p. 44)

2. If $E$ is a stable extension of $\Delta$ ($\mathcal{A}, \mathcal{C}, \preceq$), then $\bigcup_{A \in E} Prem(A)$ is a preferred subtheory of $T$.

## Implementation Notes
- ASPIC+ is not a system but a framework for specifying systems
- An argumentation system's contrariness function maps each wff to at most one set of contraries — asymmetric (contraries) or symmetric (contradictories)
- Strict rules cannot be attacked; defeasible rules can
- Arguments are recursively constructed: base case is premises from $\mathcal{K}$, inductive case applies rules to conclusions of sub-arguments
- Axiom premises ($\mathcal{K}_n$) cannot be attacked; ordinary premises ($\mathcal{K}_p$) can be
- c-consistency generalizes classical consistency via the contrariness function
- Two definitions of conflict-free: **defeat-based** (traditional, Def 13) and **attack-based** (stricter, Def 14). Paper argues attack-based is preferable — it satisfies rationality postulates more straightforwardly.
- **Rationality postulates** satisfied: sub-argument closure (Thm 12), closure under strict rules (Thm 13), direct consistency (Thm 14), indirect consistency (Thm 15).
- Section 5 introduces **weakest link** and **last link** as two standard preference orderings over arguments based on orderings over rules/premises. Paper provides a general definition of "set comparison" $\succ$ over sets of defeasible rules or premises.
- **Weakest link**: $B \prec A$ under weakest link iff: (1) if both strict, $Prem_p(B) \lhd_s Prem_p(A)$; (2) if both firm, $DefRules(B) \lhd_s DefRules(A)$; (3) otherwise both $Prem_p(B) \lhd_s Prem_p(A)$ AND $DefRules(B) \lhd_s DefRules(A)$.
- **Last link**: $B \prec A$ under last link iff: (1) $LastDefRules(B) \lhd_s LastDefRules(A)$; or (2) if both have no last defeasible rules, $Prem_p(B) \lhd_s Prem_p(A)$.
- **Eli vs Dem**: Two modes for set comparison $\lhd_s$: Elitist ($\exists X \in \Gamma$ s.t. $\forall Y \in \Gamma', X < Y$) and Democratic ($\forall X \in \Gamma, \exists Y \in \Gamma', X < Y$).
- **Key result on classical logic**: Without preferences, preferred/stable extensions of a classical-logic c-SAF correspond to maximal consistent subsets of the theory. With preferences, they correspond to Brewka's preferred subtheories (Theorem 31).
- **Attack-based vs defeat-based conflict-free**: Paper's main technical innovation. Attack-based conflict-free (Def 14) is stricter than defeat-based (Def 13). Under attack-based, all four rationality postulates hold. Under defeat-based, they hold only under additional assumptions.
- **Reasonable orderings are sufficient**: The paper identifies Definition 18 (reasonable argument orderings) as the key assumption needed for all results. Both last-link and weakest-link satisfy this when based on reasonable inducing set comparisons.
- **Preferred subtheories correspondence** (Theorem 31): Stable extensions of ASPIC+ c-SAF with classical logic correspond exactly to Brewka's preferred subtheories. This provides a formal bridge between structured and flat approaches.
- **Responses to critiques**: Section 6.2 systematically addresses Amgoud & Vesic's critiques, Brewka & Woltran's critiques. Key defense: ASPIC+ already distinguishes preference-dependent and preference-independent attacks through argument structure (sub-argument attacks), so the criticism that it conflates them is unfounded.
- **Future work** (Section 7): (1) Consider other ways of ranking arguments; (2) Study whether preference orderings satisfy the reasonableness properties; (3) Formalize ASPIC+ for real-world applications; (4) Develop structured argumentation for the Argument Interchange Format (AIF).
