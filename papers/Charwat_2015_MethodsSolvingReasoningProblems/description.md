A comprehensive survey of methods for solving reasoning problems (credulous/skeptical acceptance, extension enumeration, verification) in Dung's abstract argumentation frameworks. The paper presents and compares reduction-based approaches (SAT, CSP, ASP encodings) and direct approaches (labelling algorithms, dialogue games, dynamic programming on tree decompositions) across all standard semantics, and reviews nine implemented solver systems. Provides the formal background, complexity landscape, and algorithmic toolkit essential for implementing argumentation reasoning engines.
