# Abstract

## Original Text (Verbatim)

We first describe a metric for uncertain probabilities called opinion, and subsequently a set of logical operators on the basis of this logic reasoning with uncertain propositions. This framework which is called subjective logic uses elements from the Dempster-Shafer belief theory and we show that it is an extension of both probability calculus and binary logic.

---

## Our Interpretation

The paper addresses the fundamental problem that traditional probabilities cannot represent second-order uncertainty (how confident we are in a probability estimate). It defines opinions as (belief, disbelief, uncertainty, atomicity) tuples with a complete algebra of logical and evidential operators, proving equivalence with Beta distributions. This is directly relevant to propstore because it provides a principled alternative to bare-float confidence scores, with formal operators for combining evidence from multiple sources and propagating trust through recommendation chains.
