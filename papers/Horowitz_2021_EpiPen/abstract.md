# Abstract

## Original Text (Verbatim)

No standalone abstract is present in the paper. The introduction frames the work as a description of the logical encoding behind EpiPEn, the DSL design and implementation, and the results from applying it to six different puzzles. *(p.1)*

## Our Interpretation

EpiPEn is a Python DSL over Z3 for epistemic logic puzzles. It models what each agent knows, updates that state through public announcements and simultaneous actions, and uses simplification hints when a direct encoding would explode. The approach works well on several classic puzzles, but not on self-referential or non-linear ones. *(p.1-6)*
