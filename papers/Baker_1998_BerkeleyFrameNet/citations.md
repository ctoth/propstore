## Key Citations for Follow-up

- **Fillmore 1997 (A Frames approach to semantic analysis)**: The foundational Frame Semantics theory paper underlying the entire FrameNet project. Essential for understanding the theoretical commitments.
- **Lowe et al. 1997 (Lexical Semantics workshop paper)**: Earlier FrameNet project report from the year before — provides context on project evolution.
- **Gahl (forthcoming) — Automatic extraction of subcorpora based on subcategorization frames**: Describes the CASCADE FILTER mechanism used for subcorpus extraction, which is the automated component of the pipeline.
- **IMS 1995 Corpus Toolkit (CQP 2.1)**: The core corpus query engine underlying FrameNet's extraction pipeline — important for understanding the computational infrastructure.
