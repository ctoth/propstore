# Abstract

Bench-Capon extends Dung argumentation frameworks with values and audience-specific value orderings. In practical reasoning, an attack succeeds only when the target's promoted value is not preferred to the attacker's promoted value by the relevant audience. The paper defines value-based argumentation frameworks, audience-specific defeat, objective acceptance, subjective acceptance, and indefensibility, then uses moral-debate examples to show how persuasion can depend on value orderings and factual uncertainty.
