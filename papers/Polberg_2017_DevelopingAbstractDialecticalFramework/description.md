Polberg's PhD thesis develops Abstract Dialectical Frameworks (ADFs) as a unifying substrate for abstract argumentation, using per-argument Boolean acceptance conditions in place of Dung-style binary attack and giving a full xy-classified semantics catalogue (cc/ac/aa/ca₁/ca₂ × adm/com/prf/gr/stb/naive/sst/stg/two- and three-valued labellings) with complete P-to-Π₄ᴾ complexity results.
The second half builds an in-depth compendium of roughly ninety translations between AF, AFRA, AFN, SETAF, BAF, EAF, EAFC, EAS, and ADF — organized into a basic / coalition / attack-propagation / defender pattern catalogue, four impossibility theorems pinpointing the genuine expressivity hierarchy, and a normal-form pipeline (minimal → cleansed → weak/relation/strong validity → bypass / self-attacker consistency) that conditions when each translation is exact for which σ.
For propstore this is directly load-bearing: ADFs are a candidate target representation for the argumentation/render layer alongside the existing Dung AF and ASPIC+ paths, and the translation compendium is the reference catalog for moving claim-stance graphs between formalisms without silently changing which extensions are admissible — exactly the non-commitment discipline propstore demands at its semantic core.
