Presents FrameNet as a bridge between natural language text and structured knowledge, demonstrating how frame semantics enables inference, information extraction, question answering, and cross-lingual metaphor analysis.
Key contributions include achieving 95.9% IE annotation precision using FrameNet-compiled patterns and describing computational formalisms for frame-based inference.
The frame-based approach to preserving multiple perspectives on the same event is conceptually related to propstore's non-commitment discipline for holding rival interpretations.
