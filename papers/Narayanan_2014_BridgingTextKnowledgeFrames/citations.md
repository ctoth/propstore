## Key Citations for Follow-up

- **Fillmore, 1976** — Original Frame Semantics framework. Foundational reference for understanding the theoretical basis of all frame-based NLP work.
- **Narayanan and Harabagiu, 2004** — Frame-based question answering system. Demonstrates practical application of frame semantics to deep QA beyond factoid retrieval.
- **Mohit and Narayanan, 2003** — FrameNet-based IE pattern compilation achieving 95.9% precision. Shows how frame-annotated data can be automatically converted to extraction patterns.
- **Ruppenhofer et al., 2010** — FrameNet II extended theory and practice. The most comprehensive technical reference for FrameNet's data model and annotation methodology.
