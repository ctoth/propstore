## Surprising Findings

1. **README is stale** -- claims "57 concepts" (line 3) but filesystem has 112. Git log shows expansion session adding claims from 10 papers.

2. **Audit script found real physics errors** -- `audit_script.py` lines 262-293 document specific wrong concept mappings in LLM-extracted claims: Heisenberg claim1 maps concept22 (charge) where position q is needed, concept25 (current) where imaginary unit i is needed. Kokarev claims use concept20 (gas constant) where radius is meant. These errors are documented but not fixed.

3. **Only 2 of 6 contexts are functional** -- ctx_quantum_mechanics, ctx_statistical_mechanics, thermodynamics, and relativistic_quantum_mechanics have no `assumptions` or `excludes` fields (except relativistic_quantum_mechanics which has `parents`). They cannot participate in condition-based reasoning.

4. **The ether theory cycle is the most sophisticated demo** -- mutual attack between Maxwell and ether creates a grounded-semantics exclusion, with Cayrol derived defeat propagating to speed_of_light_from_em. This is documented in README lines 213-219 but not tested.

5. **No demo for the parent context hierarchy** -- relativistic_quantum_mechanics has `parents: [relativistic_mechanics, ctx_quantum_mechanics]` but nothing tests or demonstrates this inheritance.
