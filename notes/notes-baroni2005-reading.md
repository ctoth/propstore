### Core Idea (p.1-2)
- Proposes a general recursive schema for argumentation semantics based on decomposition along SCCs (strongly connected components)
- Introduces "SCC-recursiveness" as a property of argumentation semantics
- Motivated by limitations of preferred semantics (odd-length cycles, self-defeating arguments)

### Technical Framework (p.3-4)
- Definition 3: Given AF = (A, R), SCCs are maximal strongly connected components
- Definition 4: Restriction of AF to set S
- Definition 5: Given AF, an argument a in SCC C, defines sets D_F(a,E) — the "conditioning" from outside
- Definition 6: Base function — characterizes the recursive schema
- Definition 7: States that extensions of an AF are identified by applying the recursive base function iteratively along SCC decomposition

### Key Properties (p.2-3)
- Directionality: semantics should be determined by relevant part of AF
- SCC-recursive semantics overcome limitations of preferred semantics for odd-length cycles
- The approach generalizes beyond single semantics — provides a schema

### Worked Examples (p.4-5)
- Example 1: Unique preferred extension of AF3 (3 SCCs)
- Example 2: Preferred extension derivation
- Shows how recursive decomposition along SCCs yields extensions step by step
